User Interface overview

The User Interface (UI) is what you see when Scripture Forge is open in an Internet browser.

Here are some things you see and use in Scripture Forge, depending on your role:

  • Top App Bar

  • Navigation pane

  • Toolbar

  • Dialog boxes

  • Keyboard shortcuts

  • Choose a UI language.

  • Resize panes in the Scripture Forge window:

For example, you may want to do this when you look at questions and answers or comments below the text of a book.

    1. Move your mouse pointer over the edges between panes.

    2. If your pointer changes to be arrows with split lines (), click and drag edge of a pane to change the relative sizes of those panes in the window.
