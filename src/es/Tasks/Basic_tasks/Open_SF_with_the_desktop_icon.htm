Open Scripture Forge with the desktop icon

First time

After you install the desktop icon, do these steps:

  1. Double-click the desktop icon .

  2. Click the LOG IN button to log in.

  3. Elija un proyecto.

  4. To work offline, disconnect from the Internet.

  5. Do any tasks that do not require Internet access.

  6. When you want to stop working, click the X (Close) or press Alt+F4 to close Scripture Forge without logging out.

Important: If you Log out while offline (), none of the work that did while offline will be saved.

Normal use

  1. Double-click the desktop icon .

  2. Siga uno de estos pasos:

    • If you do not have access to the Internet and you did not log out the last time you used it, Scripture Forge opens so you can use it.

    • If you do not have access to the Internet, but you did log out the last time you used it, then you need to have access to the Internet to continue.

    • If you have access to the Internet and you are asked to log in, then log in.

Consejo

From a Chrome or Edge browser, here is an optional way to open Scripture Forge:

  1. Log in to Scripture Forge.

  2. Near the end of the web browser's address bar (also called the URL bar), click (To open this link, choose an app).

  3. In the Open with box, click Open.

Scripture Forge opens in another window. The browser tab automatically closes.
