Importar preguntas desde una hoja de cálculo

  1. Crea una hoja de cálculo:

    • Add a header row, with the "Question" and "Reference" labels in columns. 

Use one column for questions and one column for references. Any additional columns will be ignored.

  • Add one question and one reference in each row. 

For the reference, use book abbreviations that match what is used in Paratext.

For example, type MAT 1:1, not Matthew 1:1.

  • Save, export or download that spreadsheet as a CSV (*.csv) or TSV (*.tsv) file.

  1. En el panel de navegación, expanda Comprobación por la comunidad y, a continuación, haz clic en Resumen.

  2. Haga clic en el botón IMPORTAR. It is near the ADD QUESTION button.

The Import Questions dialog box appears.

  1. Click Import from spreadsheet.

  2. Navigate to the file, open it.

  3. Realice cualquiera de estos pasos en el cuadro de diálogo Importar Preguntas que aparece:

    • Limite el número de preguntas que ve con los controles Referencia de y Referencia a.

    • En la línea Filtrar preguntas, escriba las palabras para encontrar sólo las preguntas que contengan esas palabras.

    • Seleccione () Seleccione todas o varias filas individuales.

    • Haz clic en Importar Preguntas seleccionadas.

Nota

  • You can also include the book abbreviation in the file name.

  • You can import questions from TSV (Tab Delimited) (*.tsv) files, such as those from unfoldingWord.
