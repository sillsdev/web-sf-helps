Importar preguntas de Transcelerator

Administrators can import questions that are in the Transcelerator plugin.

Requisitos previos

  • Download and install Transcelerator, if it is not already installed.

  • On the Transcelerator File menu, click to select () Produce Scripture Forge Files.

Haz estos pasos:

  1. In Transcelerator, type the translations of the English questions you want to import in the Translation column.
    Then, select the Confirmed check box for those questions. Close Transcelerator.

  2. En Paratext, ejecute Enviar/Recibir el proyecto.

  3. En Scripture Forge, sincroniza el proyecto.

  4. En el panel de navegación, expanda Comprobación por la comunidad y, a continuación, haz clic en Resumen.

  5. Haga clic en el botón IMPORTAR. It is near the ADD QUESTION button.
    The Import Questions dialog box appears.

  6. Haz clic en Importar desde Transcelerator.

Consejo

  • Si editó preguntas o quiere importar más preguntas, repita estos pasos.

Temas relacionados

Resumen de las tareas del Administrador

Los administradores y cualquier persona a la que se le haya concedido permiso en la página de Usuarios pueden importar las preguntas que están en el complemento de Transcelerator.
