Configurar herramientas de proyecto

Las personas con el rol de Administrador pueden hacer esta tarea.

  • En el barra superior, haga clic en su avatar y luego haga clic en Proyectos. Entonces haga clic en Configuraciones en el Panel de navegación.

Herramientas

Traducir

  • Select () Translation Suggestions if you want to enable translation suggestions. Then, click the Based on down arrow () and click a Paratext project.

The project you choose with the Based on control is the source project. The project you are translating is the target project. These two projects can be moved (swapped) from one side of the window to another. The books in the Based on project must match the books in target project. Otherwise, when you click a book, there may not be a corresponding book in the source project.

Comprobación por la comunidad

  • Select () Community Checking if you want to enable this tool. Then, optionally, you can select () one or both of these check boxes:

    • See other's answers and comments - If selected, people will be able to see other peoples' questions and comments.

    • Share - If selected, this button (share) will appear in Community Checking.  Then people who do not have the role of Administrator can click it and then send an email invitation to others.

Then, select () one of these options:

●  Seleccione Cualquiera con un enlace si quiere permitir a cualquiera que tenga la URL del enlace del proyecto para unirse como un revisor de la comunidad.

●  Seleccione Personas específicas si quiere permitir sólo a revisores de la comunidad unirse al proyecto respondiendo a una invitación por correo electrónico.

Zona de Peligro

Nota

  • Administrators can always open Users and invite people.
