Eliminar un proyecto

  1. Elija el proyecto que va a eliminar.

  2. Haga clic en Configuración en el Panel de Navegación.

  3. Lea cuidadosamente la información que aparece a continuación eliminar este proyecto.

  4. Sincronice el proyecto con el repositorio de Paratext (recomendado).

  5. Si está seguro de que quiere continuar, pulse BORRAR ESTE PROYECTO.

The Are you sure you want to delete this project dialog box appears.

  1. Escriba el nombre del proyecto para confirmar que borrará el proyecto correcto.

  2. Haga clic en ENTIENDO LAS CONSECUENCIAS, BORRAR ESTE PROYECTO.
