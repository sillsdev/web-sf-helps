Add a comment

You can add a comment about a particular answer.

  1. Click ADD A COMMENT, which appears below a particular answer or comment.

The Your Comment box appears.

  1. Type your comment in the box.

  2. Click SAVE COMMENT.

Comments appear with the name of the person (or "Me") who added it, and a time stamp so others know who added it and when.

EDIT and DELETE also appear with your comment so you can edit or delete it.

Note

  • You cannot see an archived question or its answers or comments.
