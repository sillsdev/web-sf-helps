Edit an answer or comment

You can only edit an answer or comment that you wrote yourself.

Answer

  1. Click EDIT for the answer you want to edit.

If the answer is text, it appears in the Your Answer box.

If the answer is an audio recording, to change the audio you will have to click REMOVE AUDIO FILE first before recording a new audio answer.

  1. Do one of these steps:

    • Edit the text of the existing answer.

    • Click ANSWER and then type a new answer.

    • Click REMOVE AUDIO FILE to remove the audio file. Then, click RECORD/UPLOAD and record or upload a new answer.

  2. Click SAVE ANSWER.

Comment

  1. Click EDIT for the comment you want to edit.

The comment appears in the Your comment box.

  1. Edit the comment

  2. Click SAVE COMMENT.

Note

  • You cannot see an archived question or its answers or comments.

  • If you have no Internet access, you can edit answers even if they include audio.

Later, when you have access to the Internet, the answers and audio files are uploaded to the Scripture Forge server.

If you see an Audio unavailable message, click the question mark button () to learn why.
