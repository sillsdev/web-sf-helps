Translate tasks overview

A person with the role of Translator can do these tasks:

  • Add a comment

  • Configure translation suggestions

  • Retrain the suggestion engine

  • Swap source and target

  • Synchronize

  • Translate

Note

  • If the Editing Enabled check box is cleared () in Paratext, you cannot edit the text in Scripture Forge.

  • Note tags that came from Paratext also appear in Scripture Forge when synchronized if your role permits it.
