Synchronize

People with the role of Administrator or Translator can do these steps to synchronize this Scripture Forge project with the Paratext repository. This includes the most recent notes that appear in Paratext. You (Admin) might want to review Work with answers.

  1. Choose a project.

  2. In the Navigation pane, click Synchronize.

  3. Click the SYNCHRONIZE button in the main pane.

A progress indicator appears.

Changes made in Scripture Forge are sent to the repository. Changes made in Paratext that were sent to the repository are received in Scripture Forge.

Note

  • If you synchronize a project that is based on another project, the based-on project will also synchronize. Then, if that based-on project had been updated, you get the latest changes.

  • The progress indicator has a CANCEL button that you can click to cancel the process.

  • When reviewer leaves a note in the text in Scripture Forge, after a sync, the note in Paratext will be attributed to the Paratext user who did the sync, either an Admin or Translator. There will be not record of the Scripture Forge user who does the sync.
