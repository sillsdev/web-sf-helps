Swap source and target

If a Based on project or resource (source) was selected, you see two panes in the Scripture Forge window. One has this source and the other has your project (target).

You can change which side of the window they appear in.

  1. Choose a project.

  2. In the Navigation pane, click Translate.

  3. Click the book ( <name>).
