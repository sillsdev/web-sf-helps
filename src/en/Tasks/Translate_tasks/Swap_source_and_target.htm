Swap source and target

If you want to exchange which side of the Scripture Forge window you want to read (source) and which to type in (target), you can swap them.

  1. Choose a project.

  2. In the Navigation pane, click Translate.

  3. Click the book ( <name>).

A toolbar appears and displays the swap source and target button ().

  1. Click the button.

The projects exchange positions, from one side to the other.

Tip

  • When you translate, you type and edit in the target project. The based-on project is the source project.
