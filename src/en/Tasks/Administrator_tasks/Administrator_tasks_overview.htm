Administrator tasks overview

A person with the role of Administrator can do these tasks:

Control

  • Questions, answers or comments:

    • Add a question

      • Record or upload a question

      • Work with answers

    • Archive or republish questions

    • Connect project

    • Delete a question

    • Edit a question

    • Import questions from a spreadsheet

    • Import questions from Transcelerator

  • Project:

    • Configure project settings

    • Connect a Paratext project to Scripture Forge

    • Delete a project

    • Review translation progress

    • Synchronize

    • Users overview

View

  • Review Translation progress
