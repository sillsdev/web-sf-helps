Import questions from Transcelerator

Only people with the role of Administrator can import questions that are in the Transcelerator plugin.

Prerequisites

  • Download and install Transcelerator, if it is not already installed.

  • On the Transcelerator File menu, click to select () Produce Scripture Forge Files.

Do these steps:

  1. In Transcelerator, type the translations of the English questions you want to import in the Translation column.
    Then, select the Confirmed check box for those questions. Close Transcelerator.

  2. In Paratext, Send/Receive the project.

  3. In Scripture Forge, synchronize the project.

  4. In the Navigation pane, expand Community Checking and then click Overview.

  5. Click the Import button. It is near the Add Question button.
    The Import Questions from Transcelerator dialog box appears.

  6. Do any of these steps to limit the number of questions that appear in the dialog box:

    • Click the down arrow in the Reference from box, and then click the book, the chapter and then the verse.

    • Click the down arrow in the Reference to box, and then click the book, the chapter and then the verse.

    • Type word in the Filter questions box to filter the questions. Click SHOW ALL to see all the questions again.

  7. Select () Select All to select all the questions that appear in the dialog box. Otherwise, select the individual questions you want to import.

  8. Click IMPORT SELECTED QUESTIONS.

Tip

  • If you edited questions or want to import more questions, do these steps again.
