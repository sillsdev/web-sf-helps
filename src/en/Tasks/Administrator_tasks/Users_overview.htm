Users overview

In the Navigation pane, Users is one of the areas.

Here are things you can do in this area:

  • Share with others to invite people to help as community checkers.

  • Filter users - To find a user in the list, type their name. To filter for a particular role, type that role.

  • Can add & edit questions - For a Paratext user who is not an Administrator, select () this check box if you need to give that person permission to add, edit, import and archive questions.

  • Remove users - To remove a user from the current project, click (Remove from project).

Note

  • Users' roles and permissions can only be changed in Paratext (User permissions in Project Setting).

You will need to run Send/Receive in Paratext and then Synchronize the project from Scripture Forge.

Then, after the user logs in, their role change will be shown in Scripture Forge.
