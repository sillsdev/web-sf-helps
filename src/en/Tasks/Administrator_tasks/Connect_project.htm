Connect project

The administrator must connect the project first. After that, people with other roles can connect to it.

  1. In the Navigation pane, click the project down arrow () and then click Connect project.

The Connect Paratext Project window appears.

  1. Click the down arrow in the Paratext Project control and then click the project.

  2. If you see Additional Settings, select () or clear () check boxes as desired.

These are described in Set up project tools. These settings can be changed later.

  1. Click CONNECT.

A progress indicator appears. When is finishes, each can do their tasks appropriate for their role.

Important

  • Additional Settings appears if the project you want to connect to Scripture Forge does not exist yet in Scripture Forge.

The administrator who completes the steps in this topic will create the project in Scripture Forge.

 

After that, translators and reviewers will not see Additional Settings when they connect to the project. This is because they were set by the Administrator.
