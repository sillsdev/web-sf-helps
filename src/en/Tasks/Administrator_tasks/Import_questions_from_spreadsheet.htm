Import questions from a spreadsheet

  1. Create a spreadsheet:

    • Use two columns with one for the references and one for the questions.

    • Each row must have one reference and question. Do not use a header row.

    • Use book abbreviations that match what is used in Paratext.

    • Save or export that spreadsheet as a CSV (*.csv) file.

  2. In the Navigation pane, expand Community Checking and then click Overview.

  3. Click the BULK IMPORT button. It is near the ADD QUESTION button.
    The Import Questions dialog box appears.

  4. Click Import from CSV file.

  5. Navigate to the CSV file, click it and then click Open.

  6. Do any of these steps in the Import Questions dialog box that appears:

    • Limit the number of questions you see with the Reference from and Reference to controls.

    • In the Filter questions line, type words to find only questions that contain those words.

    • Select () Select All or various individual rows.

    • Click Import Selected Questions.
