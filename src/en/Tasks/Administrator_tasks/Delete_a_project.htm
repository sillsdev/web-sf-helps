Delete a project

  1. Choose the project you will delete.

  2. Click Settings in the Navigation pane.

  3. Carefully read the information below Delete this project.

  4. Synchronize the project with the Paratext repository (recommended).

  5. If you are sure you want to continue, click DELETE THIS PROJECT.

The Are you sure you want to delete this project dialog box appears.

  1. Type the name of the project to confirm that you will delete the correct project.

  2. Click I UNDERSTAND THE CONSEQUENCES, DELETE THIS PROJECT.
