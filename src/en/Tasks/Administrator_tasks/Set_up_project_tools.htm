Set up project tools

People with the role of Administrator can do this task.

  • On the Top App Bar, click your avatar and then click Project home. Then click Settings in the Navigation pane.

Tools

  1. If you want to disable a tool or feature, clear () its check box.

  2. Select () Translation Suggestions if you want to enable translation suggestions.

    • Then, click the Based on down arrow () and click a Paratext project.

  1. Select () Community Checking if you want to enable this tool.

Then, select () one or both of these options:

    • See other's answers and comments to allow other users to see each other peoples' questions and comments.

    • Share to enable sharing. Then, select () Anyone with a link or Specific People.

Anyone with a link enables two ways to share (Link sharing and Invite people).

Specific People means you can only invite one person (one email) at a time.

Important

  • The books that are in the Based on project must match the books in your project. Otherwise, some of your books may not appear in your project.

The project you choose with the Based on control is the source project. The project you are translating is the target project. These two projects can be moved (swapped) from one side of the window to another.

  • The Users link opens Users so you can invite people to collaborate. In Community Checking the Share button () opens the Share with others dialog box.
