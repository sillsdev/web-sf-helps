Install or remove the Scripture Forge desktop icon

Install

  1. Log in to Scripture Forge.

  2. Near the end of the web browser's address bar (also called the URL bar), click (Install Scripture Forge).

  3. In the Install app question box, click Install.

The Scripture Forge icon is installed on your desktop.

Scripture Forge opens in a separate window and the web browser automatically closes.

  1. Choose a project.

  2. Now you can disconnect your computer or device from the Internet and do other tasks that do not require access to the Internet.

Remove

  1. Double-click the Scripture Forge desktop icon .

  2. Near the top edge of the application window, click (Customize and control Scripture Forge), and then click Uninstall Scripture Forge.

  3. In the Remove "Scripture Forge" question box, click Remove.

Note

  • The steps above assume a Chrome or Edge browser on a laptop or desktop computer. You might need to get more help if your browser is different or if future upgrades to web browsers make them behave differently.

  • On many mobile devices, you can add Scripture Forge to your home screen.
