Book

In Scripture Forge, the term book is the same as in Paratext. Typically this includes those of both the Old Testament and New Testament.

Book names appear in various places:

  • In the Navigation pane, this book icon () and the book name appear together.

You see them when you choose a book.

  • In other windows or panes, another icon () can appear with the book name.

For example, you seem them when you display Community Checking areas in the Navigation pane.

  • Sometimes you just see the book name without any associated icon.

For example, the book name appears above the toolbar.
