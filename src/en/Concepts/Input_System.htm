Input System

The term input system or writing system refers to a language, its script and abbreviation, whether it is left-to-right or right-to-left, and so on.

In Scripture Forge, the when you connect a project, any input system information and versification are inherited from Paratext project properties and settings.

Currently, you need to manually select the appropriate keyboard for your language.
