Send/Receive

In Scripture Forge, people with the role of Administrator use Synchronize to update the Paratext repository with the latest changes made in Scripture Forge data, or receive the latest changes made in Paratext.

In Paratext, use Send/Receive to update Paratext with the latest changes made in Scripture Forge.
