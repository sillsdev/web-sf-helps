Keyboard shortcuts

When you translate, you can use these keyboard shortcuts:

Keys

Result

Enter

or

Ctrl + 0

 

inserts an entire phrase (inserting suggestions)

Ctrl + number key

inserts that number of suggested words, beginning with the first suggested word

Tip

  • Scripture Forge runs in an Internet browser. Browsers allow you to open a hyperlink in a new tab.

In Scripture Forge, to open an area in another tab, you can press the Ctrl key and then click an area in the Navigation pane.

Alternately, you can click an area in the Navigation pane with the middle mouse button (which is often the scroll wheel).
