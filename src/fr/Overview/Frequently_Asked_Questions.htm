Questions fréquemment posées

Puis-je... ?

  • Puis-je poser une question ? Seules les personnes ayant le rôle d'Administrateur peuvent ajouter une question. Les personnes ayant d'autres rôles peuvent ajouter une réponse ou ajouter un commentaire.

  • Ouvrir le projet dans plus d'un onglet ? Oui, sur un bureau, Scripture Forge s'exécute dans un navigateur web. Les navigateurs vous permettent d'ouvrir un hyperlien dans un nouvel onglet.

Dans Scripture Forge, pour ouvrir une zone (outil) dans un autre onglet, vous pouvez presser la touche Ctrl puis cliquer sur une zone dans le volet de navigation. Vous pouvez également cliquer sur une zone dans le volet de navigation avec le bouton de souris du milieu (qui est souvent la roulette de défilement). Ces options fonctionnent également avec tout texte de lien hypertexte que vous voyez dans l'interface utilisateur de Scripture Forge, comme ceux de la zone Paramètres.

  • Puis-je continuer à travailler sans accès à Internet () ? Oui, vous pouvez continuer à travailler avec certaines limitations.
    Voir aussi : Ouvrir Scripture Forge avec l'icône de bureau et Connexion /Déconnexion.

  • Si vous voyez un message Audio indisponible, cliquez sur le bouton point d'interrogation () pour en connaître la raison.

Comment puis-je... ?

  • Comment puis-je changer les rôles ? Les rôles de Paratext peuvent uniquement être modifiés dans Paratext. Si d'autres personnes ont été invitées à vérifier une traduction, il leur est donné le rôle de Vérificateur.

Qu'est-ce que Scripture Forge ?

  • Voir Vue d'ensemble de Scripture Forge ou https://beta.scriptureforge.org.

Qui peut... ?

  • Rôles liste les rôles utilisés dans Scripture Forge. Aperçu des tâches liste les tâches par rôle.

Pourquoi je ne peux pas... ?

  • Pourquoi je ne peux pas voir une question ? Vous ne pouvez pas voir une question archivée ou ses réponses ou commentaires.
