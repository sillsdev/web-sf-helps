Obtenir une aide supplémentaire

Voici d'autres sources d'aide que vous pouvez utiliser lorsque vous ne trouvez pas d'aide suffisante dans les sujets de l'Aide de Scripture Forge :

  • https://community.scripture.software.sil.org/c/scripture-forge

  • D'autres personnes qui utilisent Scripture Forge.

Signaler un problème

Si vous avez un problème à signaler, ou si vous souhaitez demander une fonctionnalité, procédez comme suit :

  1. Sur la barre d’application supérieure, cliquez sur le bouton Aide puis cliquez sur Signaler un problème.

Votre service d'e-mail par défaut ou les systèmes e-mail que vous utilisez apparaissent. Si ce n'est pas le cas, complétez les étapes pour ajouter un compte.

Un e-mail devrait s'ouvrir, avec l'adresse électronique (scriptureforgeissues@sil.org) déjà indiquée.

La version de Scripture Forge que vous utilisez devrait être automatiquement incluse dans l'e-mail.

L'e-mail vous indique également quels genres d'informations vous devez envoyer pour aider les développeurs.

  1. Ajoutez autant d'informations que possible.

  2. Envoyez l'e-mail.

Remarque

  • L'e-mail sera accessible au public, donc n'envoyez pas d'informations sensibles comme des mots de passe ou des informations personnelles.
