Conseil

  • Ces rubriques d'aide utilisent le verbe "cliquez", une action que vous faites en utilisant une souris.

Pour un appareil mobile, l'action équivalente serait "appuyez", une action que vous faites avec votre doigt.

  • En outre, ces rubriques d'aide utilisent le terme "bureau". Pour un appareil mobile, un terme équivalent serait "écran d'accueil" ou quelque chose de similaire.

  • Nous aurions plaisir d'entendre vos commentaires sur les Aides de Scripture Forge.
