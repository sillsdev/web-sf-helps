Question et réponse

Une question est une demande qui vise à obtenir des informations ou des éclaircissements des autres personnes.

Une réponse est une réponse à la question. Voir aussi : Commentaires.

  • Les personnes ayant le rôle d'administrateur peuvent effectuer ces tâches :

    • Ajouter des questions pour orienter et contrôler les discussions sur le travail de traduction. Les questions peuvent être saisies par écrit et/ou enregistrées en audio.

    • Archiver ou republier une question.

    • Supprimer une question.

    • Supprimer la réponse d'une autre personne.

    • Modifier une question.

    • Lire ou écouter une réponse.

  • Les personnes ayant d'autres rôles peuvent effectuer ces tâches, si elles sont activées :

    • Ajouter une réponse. Les réponses peuvent être saisies par écrit et/ou enregistrées en audio.

Les réponses des autres personnes sont cachées jusqu'à ce que vous ajoutiez votre propre réponse. Ainsi votre réponse n'est pas influencée par d'autres réponses existantes. Si vous voyez un lien qui indique Afficher 1 ou plusieurs réponses (ou similaire), cliquez dessus pour voir d'autres réponses.

    • Ajouter un commentaire.

    • Supprimer votre réponse.

    • Modifier votre réponse.

    • Mettre un J'aime sur une réponse.

    • Voir le nombre de questions, réponses, commentaires et de J'aime.

Remarques

  • Dans Vérification par la communauté, la zone Tous les questions apparaît dans le volet de navigation lorsqu'il y a des questions dans deux livres ou plus. Là, vous voyez toutes les questions, sans se limiter à un livre spécifique.

  • Les questions qui n'ont pas été lues sont en gras.

  • Le nombre actuel de réponses à chaque question apparaît près de chaque question. Exemple :

  • (point d'interrogation) apparaît dans le texte du livre qui comporte une question et ce texte est surligné.
