Envoyer/Recevoir

Dans Scripture Forge, les personnes ayant le rôle d'Administrateur utilisent Synchroniser pour mettre à jour le dépôt Paratext avec les dernières modifications apportées aux données de Scripture Forge, ou recevoir les dernières modifications apportées dans Paratext.

Dans Paratext, utilisez Envoyer/Recevoir pour mettre à jour Paratext avec les dernières modifications apportées dans Scripture Forge.
