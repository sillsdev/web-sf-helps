Commentaire

Comments with questions

Un commentaire est généralement une déclaration (remarque ou observation) concernant une réponse ou un autre commentaire. Par exemple, vous pourriez vouloir répondre aux commentaires d'une autre personne sur votre réponse afin qu'elle comprenne votre perspective.

Un commentaire pourrait également inclure une question de clarification. Par exemple, vous pouvez avoir besoin de poser une question pour mieux comprendre la réponse d'une autre personne.

  • Les personnes ayant le rôle d' administrateur peuvent effectuer ces tâches :

    • Supprimer le commentaire d'une autre personne.

  • Les personnes ayant d'autres rôles peuvent effectuer ces tâches, si activé:

    • Ajouter un commentaire.

    • Supprimer votre propre commentaire.

    • Modifier votre propre commentaire.

Comments in text

People with the role of Administrator, Translator or Commenter can add comments in the text. In this case, an icon () appears in the book, chapter and verse that has the comment. 

You can hold your mouse pointer over that icon to read the comment. On a phone or smaller device, tap the icon to open the dialog box so you can read it.
