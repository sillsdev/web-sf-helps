Vérification par la communauté

Dans Scripture Forge, l'idée de vérification par la communauté signifie simplement ce que le nom suggère. Vous pouvez lire ou écouter une traduction et la vérifier auprès de la communauté pour s'assurer de sa clarté, de son caractère naturel et de son exactitude.

Si activé, vous pouvez également effectuer d'autres tâches énumérées dans Aperçu des tâches de Vérification par la communauté.

Suivant votre rôle, voici ce que vous voyez si vous cliquez sur Vérification par la communauté puis cliquez sur Vue d'ensemble :

Tous les utilisateurs

  • - le nombre de questions dans le projet.

  • - le nombre de réponses à ces questions.

  • - le nombre de commentaires sur ces questions.

  • - le nombre de J'aime.

Remarque :

Un vérificateur voit seulement le total de ses réponses, commentaires et mentions J'aime, tandis qu'un administrateur voit le total des réponses, commentaires et mentions J'aime pour tout le monde.

Administrateurs et autres personnes ayant une autorisation

  • Les textes avec questions et le bouton AJOUTER UNE QUESTION sont affichés en haut.

En dessous de ces éléments, vous trouvez une liste de livres qui ont des questions.

Vous pouvez cliquer sur la flèche pour développer () ou réduire () les livres pour voir les questions.

    • bouton (Archive).

  • Questions archivées apparaît près du bas du volet et affiche une liste de questions qui ont été archivées.

Vous pouvez cliquer sur la flèche pour développer () ou réduire () les livres pour voir les questions.

    • bouton (Republier).

Remarque

  • Dans Vérification par la communauté, la zone Tous les questions apparaît dans le volet de navigation lorsqu'il y a des questions dans deux livres ou plus. Là, vous voyez toutes les questions, sans se limiter à un livre spécifique.

  • Les questions qui n'ont pas été lues sont en gras.
