Reformer le moteur de suggestions

Les suggestions sont disponibles si votre projet a un autre projet ou une ressource sélectionnée dans le contrôle Basé sur .

Effectuez ces étapes pour reformer le moteur de suggestion :

  1. Choisissez le projet.

  2. Dans le volet Navigation, cliquez sur Traduire.

  3. Cliquez sur la flèche () pour un livre, afin de voir les chapitres contenant des questions.

Le volet Vue d'ensemble de la traduction apparaît et affiche le volet Moteur de suggestions.

Vous voyez la qualité basée sur le nombre d'étoiles, le nombre de segments formés et le bouton REFORMER.

  1. Cliquez sur REFORMER.

Le volet Moteur de suggestions indique FORMATION et une barre de progression vous montre la progression.

Lorsque c'est terminé, vous voyez le nombre de segments formés.

Sachez que si le texte source n'a aucun livre en commun avec le projet cible, vous ne verrez aucun effet en utilisant cette fonctionnalité. Un message vous informera que les suggestions de traduction ne sont pas disponibles.

Important

  • Vous ne devriez pas le faire souvent parce qu'il apprend au fur et à mesure que vous traduisez et parce que que le processus de reformation peut prendre un certain temps.

Voici les raisons pour lesquelles vous devriez envisager de reformer le moteur de suggestions :

  • Vous avez changé de manière significative la façon dont vous traduisez quelque chose, en particulier si c'était à plusieurs endroits.

  • Vous avez effectué un nombre important de modifications suite à une vérification par un consultant.
