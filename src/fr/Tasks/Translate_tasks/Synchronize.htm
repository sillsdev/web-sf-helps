Synchronisation

Les personnes ayant le rôle d'Administrateur ou Traducteur peuvent effectuer ces étapes pour synchroniser ce projet Scripture Forge avec le dépôt Paratext. Cela inclut les remarques les plus récentes qui apparaissent dans Paratext.  Vous (Admin) voudrez peut-être vérifier Travailler avec les réponses.

  1. Choisissez un projet.

  2. Dans le volet Navigation , cliquez sur Synchroniser.

  3. Cliquez sur le bouton SYNCHRONISER dans le volet principal.

Un indicateur de progression apparaît.

Les modifications apportées dans Scripture Forge sont envoyées au dépôt. Les modifications effectuées dans Paratext qui ont été envoyées au dépôt sont reçues dans Scripture Forge.

Remarque

  • Si vous synchronisez un projet qui est basé sur un autre projet, le projet sur lequel il est basé sera également synchronisé. Ensuite, si ce projet de base a été mis à jour, vous obtenez les dernières modifications. .

  • L'indicateur de progression dispose d'un bouton ANNULER que vous pouvez cliquer pour annuler le processus.

  • When reviewer leaves a note in the text in Scripture Forge, after a sync, the note in Paratext will be attributed to the Paratext user who did the sync, either an Admin or Translator. There will be not record of the Scripture Forge user who does the sync.
