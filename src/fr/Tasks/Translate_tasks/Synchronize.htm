Synchronisation

Les personnes ayant le rôle d'Administrateur ou Traducteur peuvent effectuer ces étapes pour synchroniser ce projet Scripture Forge avec le dépôt Paratext.

  1. Choisissez un projet.

  2. Dans le volet Navigation , cliquez sur Synchroniser.

  3. Cliquez sur le bouton SYNCHRONISER dans le volet principal.

Un indicateur de progression apparaît.

Les modifications apportées dans Scripture Forge sont envoyées au dépôt. Les modifications effectuées dans Paratext qui ont été envoyées au dépôt sont reçues dans Scripture Forge.

Remarque

  • L'indicateur de progression dispose d'un bouton ANNULER que vous pouvez cliquer pour annuler le processus.
