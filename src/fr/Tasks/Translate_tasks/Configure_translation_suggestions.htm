Configurer les suggestions de traduction

  1. Choisissez un projet.

  2. Dans le volet Navigation, cliquez sur Traduire.

  3. Cliquez sur un livre ( <name>).

  1. Sur la barre d'outils cliquez sur (Configurer les suggestions de traduction).

La boîte de dialogue Paramètres de suggestions de traduction apparaît.

    • Suggestions de traduction : Cliquez sur ce contrôle pour cacher () ou afficher () les suggestions de traduction. Ce contrôle cache ou affiche des suggestions seulement pour vous dans le projet actuel.

    • Nombre de suggestions : Cliquez sur la flèche vers le bas puis cliquez sur le nombre maximum de suggestions que vous souhaitez voir.

    • Confiance dans les suggestions : Examinez la valeur en pourcentage qui apparaît au-dessus du curseur pour voir le niveau de confiance minimum des suggestions qui peuvent être proposées.

Cliquez sur le curseur de contrôle () et faites le glisser :

      • Plus augmente le nombre de suggestions en incluant plus de suggestions qui ont un faible niveau de confiance.

      • Mieux diminue le nombre de suggestions en incluant moins de suggestions qui ont un faible niveau de confiance.

  1. Cliquez sur FERMER.

Conseil

  • Le niveau de confiance d'un mot ou d'une phrase suggérée est basé sur l'utilisation de cette suggestion dans les segments.

Donc au fur et à mesure que les segments sont traduits, la précision des suggestions peut augmenter.
