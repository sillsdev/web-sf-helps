Aperçu des tâches de traduction

Une personne ayant le rôle de Traducteur peut effectuer ces tâches :

  • Configurer les suggestions de traduction

  • Reformer le moteur de suggestions

  • Intervertir la source et la cible

  • Synchronisation

  • Traduire

  • Glisser-déposer du texte

Remarque

  • Si la case Édition autorisée est décochée () dans Paratext, vous ne pouvez pas modifier le texte dans Scripture Forge.

  • Note tags that appear in Paratext also appear in Scripture Forge when synchronized.
