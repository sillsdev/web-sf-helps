Intervertir la source et la cible

Si un projet ou une ressource (source) a été sélectionné, dans Basé sur, vous voyez deux volets dans la fenêtre Scripture Forge. L'une a cette source et l'autre a votre projet (cible).

Vous pouvez modifier de quel côté de la fenêtre elle apparait.

  1. Choisissez un projet.

  2. Dans le volet Navigation, cliquez sur Traduire.

  3. Cliquez sur le livre ( <name>).
