Intervertir la source et la cible

Si vous voulez intervertir le côté de la fenêtre Scripture Forge dans lequel vous voulez lire (source) et celui dans lequel vous voulez saisir (cible), vous pouvez les permuter.

  1. Choisissez un projet.

  2. Dans le volet Navigation, cliquez sur Traduire.

  3. Cliquez sur le livre ( <name>).

Une barre d'outils apparaît et affiche le bouton intervertir la source et la cible ().

  1. Cliquez sur le bouton.

Les projets permutent leurs positions, d'un côté à l'autre.

Conseil

  • Lorsque vous traduisez, vous saisissez et modifiez dans le projet cible. Le projet sur lequel votre projet est basé est le projet source.
