Sélectionner un texte de réponse

  • Effectuez les étapes dans Ajouter une réponse jusqu'à ce que vous ayez cliqué sur SÉLÉCTIONNER UN TEXTE.

Ensuite, effectuez ces étapes.

  1. Cliquez sur SÉLECTIONNER LES VERSETS.

La boîte de dialogue Sélectionner des passages à joindre à votre réponse apparaît. Elle affiche le chapitre actuel du livre dans lequel vous travaillez. Mais, vous pouvez modifier le livre ou le chapitre.

  1. Si vous souhaitez modifier le livre ou le chapitre, cliquez sur la flèche vers le bas () qui apparaît près du nom du livre.

Ensuite, effectuez ces étapes :

    • Dans la boîte de dialogue Choisir un livre, cliquez sur le livre qui contient le texte que vous souhaitez.

    • Dans la boîte de dialogue Choisir un chapitre, cliquez sur le chapitre qui contient le texte que vous souhaitez.

    • La boîte de dialogue Sélectionner des passages à joindre à votre réponse apparaît à nouveau et affiche le livre et le chapitre.

  1. Surlignez les mots de l'une ou l'autre de ces façons :

    • Avec votre souris, cliquez et faites glisser votre curseur de souris sur le texte que vous voulez.

    • Cliquez sur un mot, puis appuyez et maintenez enfoncée la touche Shift tout en utilisant la touche flèche gauche ou droite pour surligner davantage de mots.

  2. Cliquez sur ENREGISTRER.

  3. Cliquez sur ENREGISTRER LA RÉPONSE.
