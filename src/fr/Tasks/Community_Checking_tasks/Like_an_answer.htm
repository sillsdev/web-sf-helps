Mettre un J'aime sur une réponse

  1. En ayant des réponses à une question dans l'affichage, effectuez une de ces étapes :

    • Cliquez sur pour indiquer que vous aimez une réponse.

Le nombre sous le bouton augmente de un.

    • Cliquez à nouveau sur pour indiquer que vous n'aimez pas une réponse.

Le nombre sous le bouton diminue de un.

Remarque

  • Les administrateurs ne peuvent pas mettre un J'aime sur les questions mais peuvent voir combien de J'aime ou Je n'aime pas chaque réponse a reçu.

  • Les personnes ayant d'autres rôles peuvent uniquement mettre un J'aime sur les réponses des autres personnes, pas sur les leurs.

  • Vous ne pouvez pas voir une question archivée ou ses réponses ou commentaires.
