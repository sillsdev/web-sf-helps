Modifier une réponse ou un commentaire

Vous pouvez uniquement modifier une réponse ou un commentaire que vous avez écrit vous-même.

Réponse

  1. Cliquez sur MODIFIER pour la réponse que vous souhaitez modifier.

Si la réponse est textuelle, elle apparaît dans la case Votre réponse.

Si la réponse est un enregistrement audio, pour changer l'audio, vous devrez d'abord cliquer sur SUPPRIMER LE FICHIER AUDIO avant d'enregistrer une nouvelle réponse audio.

  1. Effectuez l'une de ces étapes :

    • Modifiez le texte de la réponse existante.

    • Cliquez sur RÉPONDRE, puis saisissez une nouvelle réponse.

    • Cliquez sur SUPPRIMER LE FICHIER AUDIO pour supprimer le fichier audio. Puis cliquez sur ENREGISTRER/TÉLÉTRANSMETTRE et enregistrez ou transmettez une nouvelle réponse.

  2. Cliquez sur ENREGISTRER LA RÉPONSE.

Commentaire

  1. Cliquez sur MODIFIER pour le commentaire que vous souhaitez modifier.

Le commentaire apparaît dans la case Votre commentaire.

  1. Modifiez le commentaire.

  2. Cliquez sur ENREGISTRER LE COMMENTAIRE.

Remarque

  • Vous ne pouvez pas voir une question archivée ou ses réponses ou commentaires.

  • Si vous n'avez pas accès à Internet, vous pouvez modifier des réponses même si elles incluent de l'audio.

Plus tard, lorsque vous avez accès à Internet, les réponses et les fichiers audio sont télétransmis sur le serveur Scripture Forge.

Si vous voyez un message Audio indisponible, cliquez sur le bouton point d'interrogation () pour en connaître la raison.
