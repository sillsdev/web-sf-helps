Enregistrer ou télétransmettre une question

  • Effectuez les étapes dans Ajouter une réponse (ou Modifier une réponse) jusqu'à ce que vous ayez cliqué sur ENREGISTRER/TÉLÉTRANSMETTRE.

Ensuite, effectuez l'une des options indiquées ci-dessous.

Télétransmettre un fichier audio existant

  1. Cliquez sur TÉLÉTRANSMETTRE UN FICHIER AUDIO.

  2. Naviguez vers le fichier qui contient votre réponse, cliquez dessus et ensuite cliquez sur Ouvrir.

  3. Cliquez sur lire () et écoutez ce fichier audio.

  4. Si vous ne souhaitez pas utiliser ce fichier audio, cliquez sur SUPPRIMER LE FICHIER AUDIO. Sinon, cliquez sur ENREGISTRER LA RÉPONSE.

Enregistrer une réponse

  1. Connectez un microphone à votre ordinateur.

  2. Cliquez sur ENREGISTRER puis dites votre réponse dans votre microphone.

  3. Cliquez sur ARRÊTER L'ENREGISTREMENT.

  4. Cliquez sur lire () pour écouter votre réponse enregistrée.

  5. Si elle ne vous plaît pas, cliquez sur RÉESSAYER puis enregistrez une autre réponse. Sinon, cliquez sur ENREGISTRER LA RÉPONSE.

Remarque

  • Si vous n'avez pas accès à Internet, vous pouvez ajouter des réponses même si elles incluent de l'audio.

Plus tard, lorsque vous avez accès à Internet, les réponses et les fichiers audio sont télétransmis sur le serveur Scripture Forge.

Si vous voyez un message Audio indisponible, cliquez sur le bouton point d'interrogation () pour en connaître la raison.
