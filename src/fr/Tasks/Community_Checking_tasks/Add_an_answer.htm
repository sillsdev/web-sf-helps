Ajouter une réponse

  1. Dans le volet Navigation, cliquez sur Vérification par la communauté, puis cliquez sur une de ces zones :

    • Le livre ( <name>) qui contient la question à laquelle vous voulez répondre.

    • Toutes les questions.

Le volet Questions apparaît.

  1. Cliquez sur la question à laquelle vous voulez répondre.

La question apparaît dans le volet sous le texte du livre.

  1. Cliquez sur AJOUTER UNE RÉPONSE.

La case Votre réponse apparaît. Il y a des boutons au-dessus de cette case.

  1. Ajoutez une réponse de l'une de ces façons :

    • Cliquez sur le bouton RÉPONSE, saisissez votre réponse.

    • Cliquez sur le bouton ENREGISTRER/TÉLÉTRANSMETTRE puis effectuez les étapes dans Enregistrer ou télétransmettre une réponse.

  1. Éventuellement, cliquez sur le bouton SÉLECTIONNER LE TEXTE puis effectuez les étapes dans Sélectionner le texte de réponse, si vous souhaitez inclure le texte sélectionné avec votre réponse.

  2. Lorsque la réponse est complète, cliquez sur ENREGISTRER LA RÉPONSE.

  3. Pour passer à la question suivante, cliquez sur SUIVANT . Pour passer à une question précédente, cliquez sur PRÉCÉDENT.

Remarques

  • (point d'interrogation) apparaît dans le texte du livre qui contient une question. Ce texte est surligné et souligné.

  • Déplacez votre curseur de souris sur le bord supérieur du volet de questions et réponses jusqu'à ce qu'il se change en flèches avec des lignes de séparation (). Ensuite, cliquez et faites glisser le bord du volet pour modifier les tailles relatives des volets dans la fenêtre.

  • Après avoir enregistré votre réponse, les réponses des autres personnes apparaissent près de votre réponse.

  • Votre réponse apparaît avec votre avatar et un horodatage pour que les autres sachent qui l'a ajouté et quand il l'a fait.

  • Si vous avez ajouté le texte sélectionné, il apparaîtra avec votre réponse, généralement avec une couleur de police différente.

  • Vous pouvez modifier ou supprimer votre réponse ou ajouter un commentaire.

  • Utilisez PRÉCÉDENT () et SUIVANT () pour passer aux autres questions.

  • Vous ne pouvez pas voir une question archivée ou ses réponses ou commentaires.

  • Si vous n'avez pas accès à Internet, vous pouvez ajouter des réponses même si elles incluent de l'audio.

Plus tard, lorsque vous avez accès à Internet, les réponses et les fichiers audio sont télétransmis sur le serveur Scripture Forge.

Si vous voyez un message Audio indisponible, cliquez sur le bouton point d'interrogation () pour en connaître la raison.
