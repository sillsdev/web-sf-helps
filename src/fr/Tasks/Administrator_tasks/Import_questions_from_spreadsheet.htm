Importer des questions depuis une feuille de calcul

  1. Créer une feuille de calcul :

    • Utilisez deux colonnes avec une pour les références et une pour les questions.

    • Chaque ligne doit comporter une référence et une question. N'utilisez pas de ligne d'en-tête.

    • Utilisez des abréviations de livres qui correspondent à ce qui est utilisé dans Paratext.

    • Enregistrez ou exportez cette feuille de calcul sous forme de fichier CSV (*.csv).

  2. Dans le volet de navigation, développez Vérification par la communauté puis cliquez sur Aperçu.

  3. Cliquez sur le bouton IMPORTATION EN LOTS . Il est proche du bouton Ajouter une question.
    La boîte de dialogue Importer des questions apparaît.

  4. Cliquez sur Importer depuis un fichier CSV.

  5. Naviguez jusqu'au fichier CSV, cliquez dessus, puis cliquez sur Ouvrir.

  6. Effectuez n'importe laquelle de ces étapes dans la boîte de dialogue Importer des questions qui apparaît :

    • Limitez le nombre de questions que vous voyez avec la commande Référence de début et Référence de fin .

    • Dans la ligne Filtrer les questions, saisir des mots pour trouver uniquement les questions qui contiennent ces mots.

    • Sélectionnez () Tout sélectionner ou différentes lignes individuelles.

    • Cliquez sur Importer les questions sélectionnées.
