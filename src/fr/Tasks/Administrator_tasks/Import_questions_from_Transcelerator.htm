Importer des questions depuis Transcelerator

Seules les personnes ayant le rôle d'Administrateur peuvent importer des questions qui sont dans le plug-in Transcelerator.

Prérequis

  • Téléchargez et installez Transcelerator, s'il n'est pas déjà installé.

  • Dans le menu Fichier de Transcelerator, cliquez pour sélectionner () Produire des fichiers Scripture Forge.

Effectuez ces étapes :

  1. Dans Transcelerator, saisissez les traductions des questions en anglais que vous voulez importer dans la colonne Traduction.
    Puis, sélectionnez la case à cocher Confirmé pour ces questions. Fermez Transcelerator.

  2. Dans Paratext, effectuez un envoyer/recevoir du projet.

  3. Dans Scripture Forge, synchronisez le projet.

  4. Dans le volet de navigation, développez Vérification par la communauté puis cliquez sur Aperçu.

  5. Cliquez sur le bouton Importer. Il est proche du bouton Ajouter une question.
    La boîte de dialogue Importer des questions depuis Transcelerator apparaît.

  6. Effectuez l'une ou l'autre de ces étapes pour limiter le nombre de questions qui apparaissent dans la boîte de dialogue :

    • Cliquez sur la flèche vers le bas dans la case Début de référence, puis cliquez sur le livre, le chapitre et ensuite le verset.

    • Cliquez sur la flèche vers le bas dans la case Fin de référence, puis cliquez sur le livre, le chapitre et ensuite le verset.

    • Tapez un mot dans la case Filtrer les questions pour filtrer les questions. Cliquez sur TOUT AFFICHER pour voir à nouveau toutes les questions.

  7. Sélectionnez () Tout sélectionner pour sélectionner toutes les questions qui apparaissent dans la boîte de dialogue. Sinon, sélectionnez les questions individuelles que vous souhaitez importer.

  8. Cliquez sur IMPORTER LES QUESTIONS SÉLECTIONNÉES.

Conseil

  • Si vous avez modifié des questions ou si vous souhaitez importer plus de questions, suivez à nouveau ces étapes.
