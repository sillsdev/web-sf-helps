Importer des questions depuis Transcelerator

Administrators can import questions that are in the Transcelerator plugin.

Prérequis

  • Download and install Transcelerator, if it is not already installed.

  • On the Transcelerator File menu, click to select () Produce Scripture Forge Files.

Effectuez ces étapes :

  1. In Transcelerator, type the translations of the English questions you want to import in the Translation column.
    Then, select the Confirmed check box for those questions. Close Transcelerator.

  2. Dans Paratext, Envoyer/Recevoir le projet.

  3. Dans Scripture Forge, synchronisez le projet.

  4. Dans le volet de navigation, développez Vérification par la communauté puis cliquez sur Aperçu.

  5. Cliquez sur le bouton IMPORTATION EN LOTS . It is near the ADD QUESTION button.
    The Import Questions dialog box appears.

  6. Cliquez sur Importer depuis Transcelerator.

Conseils

  • Si vous avez modifié des questions ou si vous souhaitez importer plus de questions, suivez à nouveau ces étapes.
