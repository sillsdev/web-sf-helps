Commencer par le début

Voici la séquence que les personnes ayant le rôle d'Administrateur vont probablement suivre pour paramétrer un projet afin que les personnes ayant d'autres rôles puissent traduire et vérifier des livres.

  1. Allez sur https://beta.scriptureforge.org (pour la version Beta V2) et ensuite connectez-vous.

  2. Connectez Scripture Forge à un projet Paratext.

  3. Effectuez ces tâches si nécessaire :

    • Configurer les outils du projet.

    • Vérifier les informations dans Partager avec d'autres. Si nécessaire, suivre les étapes de cette rubrique d'aide.

    • Ajouter une question.

    • Toute autre tâche listée dans Vue d'ensemble des tâches de l'administrateur.
