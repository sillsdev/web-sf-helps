Ajouter une question

Les administrateurs et toute personne ayant reçu les autorisations sur la page Utilisateurs peuvent ajouter des questions.

  1. Cliquez sur l'une de ces icônes dans le volet de navigation :

    • nom du livre.

Ensuite, cliquez sur (Ajouter une question).

C'est la méthode recommandée car elle ouvre la nouvelle question en ayant déjà renseigné la référence biblique.

Cela peut vous faire gagner beaucoup de temps lorsque vous ajoutez plusieurs questions au même livre et au même chapitre.

    • Vérification par la communauté
      Ensuite, cliquez sur Aperçu puis cliquez sur AJOUTER UNE QUESTION dans la fenêtre Textes avec questions.

    • Toutes les questions
      Puis, cliquez sur (Ajouter une question).

La boîte de dialogue Nouvelle question s'ouvre.

  1. Suivez ces étapes dans la boîte de dialogue :

    • Boîte référence biblique - cliquez sur la flèche vers le bas, cliquez sur un livre, cliquez sur un chapitre, et ensuite cliquez sur un verset.

    • Boîte référence de fin - cliquez sur la flèche vers le bas, cliquez sur un livre, cliquez sur un chapitre, et ensuite cliquez sur un verset.

    • Boîte question - tapez ou collez votre question.

    • Si vous ajouterez une question audio, voir Enregistrer ou télétransmettre une question.

  1. Cliquez sur ENREGISTRER.

Remarque

  • Toutes les questions apparaît quand plus d'un livre a une question.

  • Si vous n'avez pas d'accès à Internet, vous pouvez ajouter des questions même si elles incluent de l'audio. L'audio est stocké sur votre appareil ainsi vos questions sont disponibles pendant que vous n'avez aucun accès à Internet. .

Plus tard, lorsque vous avez accès à Internet, les questions et les fichiers audio sont télétransmis sur le serveur Scripture Forge. Après cela, vous pouvez écouter les questions et les réponses des autres personnes.
