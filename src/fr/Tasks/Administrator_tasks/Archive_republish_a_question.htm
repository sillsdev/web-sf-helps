Archiver ou republier des questions

Les personnes ayant le rôle d'Administrateur peuvent archiver une question et, plus tard, choisir de la republier. Vérification par la communauté décrit les contrôles.

Archiver

  1. Dans le volet Navigation, cliquez sur Vérification par la communauté.

  2. Effectuez l'une de ces étapes :

    • Cliquez sur le livre () qui contient la question que vous voulez archiver. Dans la colonne Questions, cliquez sur la question. Ensuite, sous le texte du livre, cliquez sur ARCHIVER.

    • Cliquez sur Aperçu. Dans le volet Textes avec questions. Cliquez sur la flèche () pour un livre, afin de voir les chapitres contenant des questions. Cliquez sur la flèche () pour un chapitre, afin de voir les questions dans ce chapitre. Cliquez sur le bouton archiver () qui apparaît à côté de la question que vous voulez archiver.

La question et toutes les réponses et les commentaires ne peuvent plus être lus par les autres utilisateurs. Dans Aperçu, la question passe du volet Textes avec questions au volet Questions archivées.

Republier

  1. Dans le volet Navigation, cliquez sur Vérification par la communauté.

  2. Cliquez sur Aperçu.

Le volet Questions archivées apparaît. Les livres qui ont des questions archivées apparaissent en dessous.

  1. Cliquez sur la flèche () pour un livre, afin de voir les chapitres contenant des questions.

  2. Cliquez sur la flèche () pour un chapitre, afin de voir les questions dans ce chapitre.

  3. Cliquez sur le bouton republier () qui apparaît à côté de la question que vous ne voulez pas dans les archives.

Cette question sort du volet Questions archivées pour revenir au volet Textes avec questions.
