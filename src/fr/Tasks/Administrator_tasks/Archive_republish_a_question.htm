Archiver ou republier des questions

  • Dans le volet Navigation, cliquez sur Vérification par la communauté.

Archiver

  • Pour archiver une question pendant que le texte du livre est affiché, procédez comme suit :

  1. Cliquez sur le livre () qui contient la question que vous voulez archiver.

  2. Effectuez l'une de ces étapes :

  3. Sous le texte et les questions du livre, cliquez sur Archiver.

  • Pour archiver une ou plusieurs questions, procédez comme suit :

  1. Dans le volet Navigation, cliquez sur Vérification par la communauté.

  2. Dans le volet Gérer les questions, cliquez sur la flèche () d'un livre pour voir les chapitres contenant des questions. Ensuite, cliquez sur la flèche () pour un chapitre pour voir toutes les questions dans ce chapitre.

  3. Cliquez sur le bouton archiver () qui apparaît dans la ligne livre pour archiver toutes les questions du livre, ou dans la ligne chapitre pour archiver toutes les questions d'un chapitre particulier, ou dans la ligne question pour n'archiver qu'une seule question.

  4. Cliquez surArchiver dans la zone de questions qui apparaît lorsque vous archivez toutes les questions d'un livre ou d'un chapitre.

La question et toutes les réponses et les commentaires ne peuvent plus être lus par les autres utilisateurs. Dans Vue d'ensemble, la ou les questions passent du volet Gérer les questions au volet Questions archivées.

Republier

  1. Cliquez sur la flèche () pour un livre, afin de voir les chapitres contenant des questions.

The Archived Questions pane appears at the bottom of the page. Les livres qui ont des questions archivées apparaissent en dessous.

  1. Cliquez sur la flèche () pour un livre, afin de voir les chapitres contenant des questions.

  2. Cliquez sur la flèche () pour un chapitre, afin de voir les questions dans ce chapitre.

  3. Cliquez sur le bouton republier () qui apparaît dans l'une de ces lignes :

  • La ligne pour le livre si vous allez republier toutes les questions dans ce livre.

  • La ligne pour un chapitre particulier que vous voulez republier.

  • La ligne pour la question spécifique que vous voulez republier.

  1. Cliquez sur Republier dans la zone de questions, si elle apparaît.

That question or questions move from the Archived Questions pane back to the Manage questions pane. Then, they will be available for community checkers to see and answer again.
