Supprimer un projet

Les personnes ayant le rôle d'administrateur peuvent effectuer cette tâche.

  1. Choisissez le projet que vous allez supprimer.

  2. Cliquez sur Paramètres dans le volet de navigation.

  3. Lisez attentivement les informations en-dessous de Supprimer ce projet.

  4. Synchronisez le projet avec le dépôt Paratext (recommandé).

  5. Si vous êtes sûr de vouloir continuer, cliquez sur SUPPRIMER CE PROJET.

La boîte de dialogue Êtes- vous sûr de vouloir supprimer ce projet apparaît.

  1. Tapez le nom du projet pour confirmer que vous allez supprimer le bon projet.

  2. Cliquez sur JE COMPRENDS LES CONSÉQUENCES, SUPPRIMER CE PROJET.
