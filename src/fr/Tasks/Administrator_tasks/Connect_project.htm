Connecter un projet

L'administrateur doit d'abord connecter le projet. Après cela, les personnes ayant d'autres rôles peuvent s'y connecter.

  1. Dans le volet Navigation, cliquez sur la flèche vers le bas pour le projet () et ensuite, cliquez sur Connecter un projet.

La fenêtre Connecter un projet Paratext apparaît.

  1. Cliquez sur la flèche vers le bas dans la commande Projet Paratext puis cliquez sur le projet.

  2. Si vous voyez Paramètres supplémentaires, sélectionnez () ou décochez () les cases à cocher comme souhaité.

Celles-ci sont décrites dans Configurer les outils du projet. Ces paramètres peuvent être modifiés plus tard.

  1. Cliquez sur CONNECTER.

Un indicateur de progression apparaît. Quand c'est terminé, chacun peut effectuer les tâches appropriées à son rôle.

Important

  • Paramètres supplémentaires apparaît si le projet que vous voulez connecter à Scripture Forge n'existe pas encore dans Scripture Forge.

L'administrateur qui termine les étapes décrites dans cette rubrique créera le projet dans Scripture Forge.

 

Après cela, les traducteurs et les relecteurs ne verront pas Paramètres supplémentaires lorsqu'ils se connecteront au projet. C'est parce que ceux-ci ont été définis par l'administrateur.
