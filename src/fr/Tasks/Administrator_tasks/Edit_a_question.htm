Modifier une question

Les administrateurs et toute personne ayant reçu les autorisations sur la page Utilisateurs peuvent modifier des questions.

  1. Dans le volet Navigation, cliquez sur Vérification par la communauté.

  2. Effectuez l'une de ces étapes pour ouvrir la boîte de dialogue Modifier une question :

    • Cliquez sur le livre () qui contient la question que vous voulez modifier. Effectuez l'une de ces étapes : Ensuite, sous le texte du livre, cliquez sur MODIFIER.

    • Cliquez sur la flèche () pour un livre, afin de voir les chapitres contenant des questions. Dans le volet Gérer les questions, cliquez sur la flèche () pour un livre pour voir les chapitres contenant des questions. Cliquez sur la flèche () pour un chapitre, afin de voir les questions dans ce chapitre. Cliquez sur le crayon d'édition () qui apparaît à côté de la question que vous voulez modifier.

Si la question a au moins une réponse, une boîte de question apparaît. Si vous êtes sûr que vos modifications n'affecteront pas les réponses existantes, cliquez sur MODIFIER QUAND MÊME.

La boîte de dialogue Modifier une question apparaît.

  1. Choisissez une autre Référence biblique () ou Référence de fin () ou modifiez la question dans la case Question.

  2. Cliquez sur ENREGISTRER.

Important

  • Si votre modification peut avoir un impact sur la façon dont les personnes répondraient à votre question, et rendre les réponses existantes invalides, ajoutez une nouvelle question.

Si votre modification sert uniquement à corriger une faute de frappe ou une autre erreur mineure, alors MODIFIER QUAND MÊME peut être une option sûre.

  • Si vous n'avez pas d'accès à Internet, vous pouvez modifier des questions même si elles incluent de l'audio.

Plus tard, lorsque vous avez accès à Internet, les questions et les fichiers audio sont télétransmis sur le serveur Scripture Forge.
