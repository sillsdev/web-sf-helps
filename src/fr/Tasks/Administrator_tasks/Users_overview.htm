Aperçu des utilisateurs

Dans le volet Navigation, une des zones est Utilisateurs.

Voici ce que vous pouvez faire dans cette zone :

  • Partager avec d'autres.

  • Filtrer les utilisateurs - Pour trouver un utilisateur dans la liste, tapez son nom. Pour filtrer un rôle particulier, tapez ce rôle.

  • Pour supprimer un utilisateur du projet actuel, cliquez sur .

Remarque

  • Les rôles et autorisations des utilisateurs peuvent uniquement être changés dans Paratext (Utilisateurs, rôles et autorisations dans Paramètres du projet).

Vous devrez exécuter Envoyer/Recevoir dans Paratext et ensuite Synchroniser le projet depuis Scripture Forge.

Ensuite, après que l'utilisateur se soit connecté, son changement de rôle sera affiché dans Scripture Forge.
