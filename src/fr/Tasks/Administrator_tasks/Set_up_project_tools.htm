Configurer les outils du projet

Les personnes ayant le rôle d'administrateur peuvent effectuer cette tâche.

  • Sur la barre d’application supérieure, cliquez sur votre avatar puis cliquez sur Accueil du projet. Puis cliquez sur Paramètres dans le volet Navigation.

Outils

Traduire

  • Sélectionnez () Suggestions de traduction si vous souhaitez activer les suggestions de traduction. Ensuite, cliquez sur la flèche vers le bas Basé sur () et cliquez sur un projet Paratext.

Le projet que vous choisissez avec le contrôle Basé sur est le projet source. Le projet que vous traduisez est le projet cible. Ces deux projets peuvent être déplacés (échangés) d'un côté à l'autre de la fenêtre. Les livres dans le projet de base doivent correspondre aux livres dans le projet cible. Sinon, lorsque vous cliquez sur un livre, il se peut qu'il n'y ait pas de livre correspondant dans le projet source.

Vérification par la communauté

  • Sélectionnez () Vérification par la communauté si vous voulez activer cet outil. Ensuite, si vous le souhaitez, vous pouvez sélectionner () une des deux cases à cocher ou les deux :

    • Voir les réponses et commentaires des autres- Si sélectionné, les personnes pourront voir les questions et commentaires des autres personnes.

    • Partager - Si sélectionné, ce bouton (partager) apparaîtra dans Vérification par la communauté.  Puis les personnes qui n'ont pas le rôle d'Administrateur peuvent cliquer dessus et ensuite envoyer une invitation par e-mail à d'autres.

Ensuite, sélectionnez () une de ces options :

●  Sélectionnez Toute personne ayant le lien si vous voulez autoriser toute personne ayant l'URL du lien du projet à se joindre en tant que vérificateur de la communauté.

●  Sélectionnez Personnes spécifiques si vous voulez autoriser les vérificateurs de la communauté à se joindre au projet uniquement en répondant à une invitation envoyée par e-mail.

Remarque

  • Les administrateurs peuvent toujours ouvrir Utilisateurs et inviter des personnes.
