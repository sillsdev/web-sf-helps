Configurer les outils du projet

Les personnes ayant le rôle d'administrateur peuvent effectuer cette tâche.

  • Sur la barre d’application supérieure, cliquez sur votre avatar puis cliquez sur Accueil du projet. Puis cliquez sur Paramètres dans le volet Navigation.

Outils

  1. Si vous souhaitez désactiver un outil ou une fonctionnalité, décochez () sa case.

  2. Sélectionnez () Suggestions de traduction si vous souhaitez activer les suggestions de traduction.

    • Ensuite, cliquez sur la flèche vers le bas Basé sur () et cliquez sur un projet Paratext.

  1. Sélectionnez () Vérification par la communauté si vous voulez activer cet outil.

Ensuite, sélectionnez () une de ces options ou les deux :

    • Voir les réponses et commentaires des autres pour permettre à chaque utilisateur de voir les questions et commentaires des autres personnes.

    • Partager pour activer le partage. Ensuite, sélectionnez () Toute personne ayant le lien ou Personnes spécifiques.

Toute personne ayant le lien permet deux façons de partager (Partage de lien et Inviter des personnes).

Personnes Spécifiques signifie que ne pouvez inviter qu'une seule personne (un seul e-mail) à la fois.

Important

  • Les livres qui sont dans le projet de base doivent correspondre aux livres de votre projet. Sinon, certains de vos livres risquent de ne pas apparaître dans votre projet.

Le projet que vous choisissez avec le contrôle Basé sur est le projet source. Le projet que vous traduisez est le projet cible. Ces deux projets peuvent être déplacés (échangés) d'un côté à l'autre de la fenêtre.

  • Le lien Utilisateurs ouvre Utilisateurs afin que vous puissiez inviter des personnes à collaborer. Dans Vérification par la communauté le bouton Partager () ouvre la boîte de dialogue Partager avec d'autres.
