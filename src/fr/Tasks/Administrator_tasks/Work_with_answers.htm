Travailler avec les réponses

Prérequis

  • Examinez la sélection pour Exportation des réponses dans Paramètres.

Si Toutes les réponses ou Aucune réponse est sélectionné (), vous n'avez pas besoin de marquer une réponse pour l'exportation.

Toutefois, vous pouvez cliquer sur Marquer pour exportation pour les questions, et elles seront disponibles si vous sélectionnez Réponses marquées pour exportation ultérieurement.

Effectuez ces étapes :

  1. Dans le volet Navigation, cliquez sur Vérification par la communauté, puis cliquez sur une de ces zones :

    • Le livre ( <name>) qui contient la question avec la réponse que vous voulez voir.  

    • Toutes les questions.

Le volet Questions s'affiche.

  1. Cliquez sur la question qui contient les réponses que vous voulez voir.

  2. Effectuez l'une de ces étapes pour chaque réponse si nécessaire :

  • Cliquez sur Marquer pour exportation si vous souhaitez que la réponse soit exportée vers Paratext sous forme de remarque lorsque le projet est synchronisé.

  • Cliquez sur Résoudre une réponse si vous voulez résoudre la remarque correspondante dans Paratext lorsque le projet est synchronisé.

  • Pour supprimer une réponse, cliquez sur Supprimer, puis sur Supprimer dans la boîte de question qui s'affiche.

Conseils

  • Les options Exportation des réponses dans Paramètres et les options Marquer pour exportation et Résoudre que vous voyez avec chaque réponse vous permettent cette flexibilité :

Vous pouvez facilement choisir quelles réponses seront exportées vers Paratext en tant que remarques, et quand cela se produit, et quelles remarques seront résolues dans Paratext.

Lorsque vous cliquez sur Marquée pour exportation ou Résoudre, le bouton change de couleur pour cette réponse. Vous pourrez ainsi voir rapidement quelles sont les réponses que vous devez encore considérer. Sujets Associés
