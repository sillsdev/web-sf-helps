Vérifier l'avancement de la traduction

  1. Choisissez le projet.

  2. Dans le volet Navigation, cliquez sur Traduire.

  3. Cliquez sur la flèche () pour un livre, afin de voir les chapitres contenant des questions.

The Translate Overview pane appears.

Below Progress is a list of all the books in the project.

Below each book name is the number of segments that have a translation and total number of segments.

The circular progress bar () offers a graphical representation of those numbers.
