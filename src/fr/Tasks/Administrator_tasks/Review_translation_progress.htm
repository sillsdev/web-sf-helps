Vérifier l'avancement de la traduction

Les personnes ayant le rôle d'administrateur peuvent effectuer cette tâche.

  1. Choisissez le projet.

  2. Dans le volet Navigation, cliquez sur Traduire.

  3. Cliquez sur Aperçu.

Le volet Vue d'ensemble de la traduction apparaît.

Ci-dessous Avancement est une liste de tous les livres dans le projet.

En dessous de chaque nom de livre se trouve le nombre de segments qui ont une traduction et le nombre total de segments.

La barre d'avancement circulaire () offre une représentation graphique de ces chiffres.
