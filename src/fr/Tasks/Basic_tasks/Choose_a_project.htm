Choisir un projet

  1. Sur la barre d’application supérieure, cliquez sur votre avatar, puis cliquez sur Accueil du projet.

Le projet actuel, le cas échéant, est affiché en haut du volet de Navigation.

  1. Pour choisir un autre projet, cliquez sur la flèche vers le bas () puis cliquez sur un projet de la liste.

Vous pouvez maintenant cliquer sur Traduire ou Vérification par la communauté pour choisir un livre ou utiliser d'autres commandes (suivant le rôle).

Si le projet que vous souhaitez n'est pas connecté, contactez l'administrateur de votre projet.
