Partager avec d'autres

Vous pouvez inviter des personnes à être vérificateurs.

  1. Effectuez une de ces étapes dans le volet Navigation :

    • Si votre rôle est Administrateur, cliquez sur Utilisateurs.

Les contrôles Partager avec d'autres apparaissent.

    • Pour tous les autres rôles, cliquez sur Vérification par la communauté puis choisissez un livre. Cliquez sur (Partager) sur la barre d'outils.

La boîte de dialogue Partager avec d'autres apparait.

  1. Effectuez une de ces étapes comme souhaité ou comme activé :

    • Partage de lien : Si vous voulez envoyer un lien aux personnes, cliquez sur pour le copier dans votre presse-papiers. Cliquez sur TERMINÉ. Ensuite, envoyez ce lien par e-mail aux personnes que vous souhaitez inviter.

    • Inviter des personnes : Dans la case E-mail, saisissez ou collez l'adresse e-mail de la personne que vous souhaitez inviter comme utilisateur, puis cliquez sur ENVOYER. Cliquez sur TERMINÉ.

Important

  • Une fois le projet connecté, tout autre utilisateur Paratext de ce projet peut s'y connecter dans Scripture Forge. Cependant, les personnes qui feront la Vérification par la communauté ne sont typiquement pas des utilisateurs de Paratext et doivent être invitées.

La personne invitée doit accepter l'invitation, fournir un mot de passe ou se connecter avec d'autres identifiants tels que Google ou Facebook. Cette personne se voit attribuer le rôle de Vérificateur.

  • Partage de liens est disponible lorsque Toute personne ayant le lien est sélectionné () dans Paramètres.
