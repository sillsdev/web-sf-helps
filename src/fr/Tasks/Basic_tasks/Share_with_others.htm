Partager avec d'autres

Les administrateurs peuvent toujours inviter d'autres personnes à partager le travail dans le projet. Si l'administrateur a activé la fonction Partager (), alors d'autres utilisateurs peuvent effectuer ces étapes :

  1. Dans le volet Navigation, cliquez sur un livre sous Vérification par la communauté.

  2. Cliquez sur sur la barre d'outils.

La boîte de dialogue Partager avec d'autres apparait.

Partage de lien n'apparaît que si l'administrateur l'a activé.

  1. Effectuez une de ces étapes comme souhaité ou comme activé :

    • Partage de lien : Si vous voulez envoyer un lien aux personnes, cliquez sur pour le copier dans votre presse-papiers. Cliquez sur TERMINÉ.

Vous pouvez maintenant coller ce lien dans un SMS ou une message de conversation, le poster sur Facebook ou dans un e-mail, etc. D'autres personnes peuvent également partager ce lien.

    • Inviter des personnes : Dans la case E-mail, saisissez ou collez l'adresse e-mail de la personne que vous souhaitez inviter comme utilisateur. Choisissez un rôle de Scripture Forge. Choisissez une langue pour l'e-mail d'invitation. Cliquez sur ENVOYER. Cliquez sur TERMINÉ.

Le lien qui est envoyé est spécifiquement pour la personne qui a reçu l'e-mail, il ne peut donc pas être utilisé par quelqu'un d'autre.

Important

  • Si Vérification par la communauté n'est pas sélectionné () dans Paramètres, le seul rôle Scripture Forge disponible est Voir la traduction.

  • Une fois le projet connecté, tout autre utilisateur Paratext de ce projet peut s'y connecter dans Scripture Forge. Cependant, les personnes qui feront la Vérification par la communauté ne sont typiquement pas des utilisateurs de Paratext et doivent être invitées. Dans ce cas, Vérification par la communauté doit être sélectionné ().

  • La personne invitée doit s'inscrire pour obtenir un compte si elle n'en a pas déjà un.

  • Les administrateurs peuvent effectuer ces mêmes étapes dans Utilisateurs.
