Choisir une langue d'interface utilisateur

  • Sur la barre d’application supérieure, cliquez sur (Langue) puis cliquez sur la langue que vous souhaitez.

L'interface utilisateur (IU) passe à cette langue. Si certains mots n'ont pas été traduits, ils apparaîtront en anglais.

Si vous ne voyez pas la langue que vous souhaitez dans la liste, obtenez plus d'aide.
