Lire ou écouter une réponse

Les personnes ayant le rôle d'administrateur peuvent lire ou écouter les réponses des autres. Les personnes ayant d'autres rôles peuvent uniquement lire ou écouter les réponses après avoir ajouté une réponse elles-mêmes.

  1. Dans le volet Navigation, cliquez sur Vérification par la communauté.

  2. Cliquez sur le livre ( <name>) qui contient les réponses aux questions.

Le volet Questions apparaît à côté du volet avec le texte du livre.

  1. Cliquez sur la question que vous voulez voir.

Un volet apparaît sous le texte du livre. Il affiche la question et en dessous vous voyez les commentaires et les réponses.

Si nécessaire, cliquez et faites glisser le séparateur () entre les volets pour les redimensionner ou utilisez la barre de défilement.

  1. Voici les choses que vous pouvez faire, suivant votre rôle et si vous avez ou non écrit la question, la réponse ou les commentaires :

    • Lire une réponse qui a été saisie.

    • Cliquer sur le bouton lire () pour écouter une réponse qui a été enregistrée.

    • Cliquer sur AJOUTER UN COMMENTAIRE pour ajouter un commentaire.

    • Cliquer sur MODIFIER pour modifier une question ou un commentaire.

    • Cliquer sur SUPPRIMER pour supprimer une question, une réponse ou un commentaire.

    • Cliquer sur pour mettre un J'aime sur une réponse.

    • Cliquer sur PRÉCÉDENT ou SUIVANT pour voir les autres questions.

Remarque

  • Si vous voyez un message Audio indisponible, cliquez sur le bouton point d'interrogation () pour en connaître la raison.
