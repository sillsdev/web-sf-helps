Installer ou supprimer l'icône de bureau Scripture Forge

Installer

  1. Connectez-vous à Scripture Forge.

  2. Vers la fin de la barre d'adresse du navigateur web (également appelée barre d'URL), cliquez sur (Installer Scripture Forge).

  3. Dans la zone question de Installer l'app, cliquez sur Installer.

L'icône Scripture Forge est installée sur votre bureau.

Scripture Forge s'ouvre dans une fenêtre séparée et le navigateur web se ferme automatiquement.

  1. Choisissez un projet.

  2. Vous pouvez maintenant déconnecter votre ordinateur ou appareil d'Internet et effectuer d'autres tâches qui ne nécessitent pas d'accès Internet.

Supprimer

  1. Double-cliquez sur l'icône de bureau Scripture Forge .

  2. Près du bord supérieur de la fenêtre d'application, cliquez sur (Personnaliser et contrôler Scripture Forge), puis cliquez sur Désinstaller Scripture Forge.

  3. Dans la zone question de Supprimer "Scripture Forge", cliquez sur Supprimer.

Remarque

  • Les étapes ci-dessus partent du principe qu'un navigateur Chrome ou Edge est installé sur un ordinateur portable ou de bureau. Vous pouvez avoir besoin d'obtenir une aide supplémentaire si votre navigateur est différent ou si de futures mises à jour des navigateurs web les amènent à se comporter différemment.

  • Sur de nombreux appareils mobiles, vous pouvez ajouter Scripture Forge à votre écran d'accueil.
