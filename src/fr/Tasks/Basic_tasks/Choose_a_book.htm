Choisir un livre

  1. Choisissez le projet qui contient le livre avec lequel vous voulez travailler.

  2. Dans le volet Navigation, cliquez sur Traduire ou Vérification par la communauté.

  3. Effectuez l'une de ces étapes :

    • Cliquez sur le livre souhaité ( <name>).

    • Dans Aperçu, cliquez sur le nom du livre que vous souhaitez.

  4. Utilisez la barre d'outils pour choisir un chapitre.
