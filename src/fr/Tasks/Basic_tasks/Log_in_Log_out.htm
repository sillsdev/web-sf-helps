Se connecter / se déconnecter

Se connecter

  • Après votre inscription, et une fois le projet connecté, cliquez sur Connexion et effectuez l'une de ces étapes :

    • Cliquez sur SE CONNECTER AVEC PARATEXT, puis suivez les étapes dans les pages Autoriser la demande.

Votre rôle dans le projet Paratext est le même dans Scripture Forge si vous avez utilisé SE CONNECTER AVEC PARATEXT au moins une fois.

Après cela, votre rôle dans le projet est gardé en mémoire si vous vous connectez avec les autres options de connexion ci-dessous.

  • Cliquez sur SE CONNECTER AVEC GOOGLE, puis suivez les étapes dans les pages Se connecter avec Google.

  • Cliquez sur SE CONNECTER AVEC FACEBOOK, puis suivez les étapes dans les pages Se connecter avec Facebook.

  • Cliquez sur SE CONNECTER AVEC UN NUMÉRO DE TÉLÉPHONE.

  • Si l'indicatif du pays que vous voyez est correct, saisissez votre numéro de téléphone.

  • Sinon, cliquez sur la flèche () et recherchez l'indicatif de votre pays. Saisissez ensuite votre numéro de téléphone.

Un code est envoyé sur votre téléphone.

Saisissez ce code pour vous connecter.

  • Saisissez votre adresse e-mail et votre mot de passe dans les cases, puis cliquez sur SE CONNECTER.

Déconnexion

  • Pour vous déconnecter, sur la barre d’application supérieure, cliquez sur votre avatar puis cliquez sur se déconnecter.

Important

  • Toutes les données Scripture Forge sont supprimées de votre ordinateur ou appareil à chaque fois que vous vous déconnectez. Donc, si vous voulez enregistrer les modifications que vous avez apportées pendant que vous étiez hors connexion () ne vous déconnectez pas avant d'avoir une connexion internet ().
    Les modifications apportées pendant que vous êtes hors ligne sont stockées localement et seront envoyées au serveur la prochaine fois que vous aurez une connexion internet. Cependant, ce stockage local est effacé à chaque fois que vous vous déconnectez. La déconnexion ne supprime pas les données du serveur.
