Ouvrir Scripture Forge avec l'icône de bureau

Première utilisation

Après avoir installé l'icône de bureau, suivez les étapes suivantes :

  1. Double-cliquez sur l'icône de bureau .

  2. Cliquez sur le bouton CONNECTER pour vous connecter.

  3. Choisissez un projet.

  4. Pour travailler hors connexion, déconnectez-vous d'Internet.

  5. Effectuez toutes les tâches qui ne nécessitent pas d'accès à Internet.

  6. Lorsque vous voulez arrêter de travailler, cliquez sur le X (Fermer) ou appuyez sur Alt F4 pour fermer Scripture Forge sans déconnexion.

Important : Si vous vous déconnectez lorsque vous êtes hors connexion (), aucun travail effectué hors connexion ne sera enregistré.

Utilisation normale

  1. Double-cliquez sur l'icône de bureau .

  2. Effectuez l'une de ces étapes :

    • Si vous n'avez pas accès à Internet et que vous ne vous êtes pas déconnecté la dernière fois que vous l'avez utilisé, Scripture Forge s'ouvre pour que vous puissiez l'utiliser.

    • Si vous n'avez pas accès à Internet, mais que vous vous êtes déconnecté la dernière fois que vous l'avez utilisé, alors vous devez avoir accès à Internet pour continuer.

    • Si vous avez accès à Internet et que vous êtes invité à vous connecter, alors connectez-vous.

Conseils

Depuis un navigateur Chrome ou Edge, voici une autre manière d'ouvrir Scripture Forge :

  1. Connectez-vous à Scripture Forge.

  2. Vers la fin de la barre d'adresse du navigateur web (également appelée barre d'URL), cliquez sur (Pour ouvrir ce lien, choisir une application).

  3. Dans la case Ouvrir avec, cliquez sur Ouvrir.

Scripture Forge s'ouvre dans une autre fenêtre. L'onglet du navigateur se ferme automatiquement.
