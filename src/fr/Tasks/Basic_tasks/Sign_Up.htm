Inscription

  1. Dans un navigateur web, allez sur https://scriptureforge.org/.

Ou, si vous avez reçu un e-mail Invitation pour rejoindre un projet, cliquez sur l'URL du lien du projet dans cet e-mail.

  1. Cliquez sur Inscription.

Effectuez l'une de ces étapes :

    • Cliquez sur INSCRIPTION VIA PARATEXT.

Pour les personnes disposant d'un compte Paratext, c'est la méthode recommandée pour s'inscrire.

Si vous n'êtes pas inscrit à Paratext, cliquez ici pour vous inscrire gratuitement.

Les personnes ayant le rôle d'Administrateur doivent avoir un compte Paratext.

    • Cliquez sur INSCRIPTION VIA GOOGLE.

    • Cliquez sur INSCRIPTION VIA FACEBOOK, puis suivez les étapes dans les pages Se connecter avec Facebook.

    • Cliquez sur SE CONNECTER AVEC UN NUMÉRO DE TÉLÉPHONE.

  • Si l'indicatif du pays que vous voyez est correct, saisissez votre numéro de téléphone.

  • Sinon, cliquez sur la flèche () et recherchez l'indicatif de votre pays. Saisissez ensuite votre numéro de téléphone.

Un code est envoyé sur votre téléphone.

Saisissez ce code pour vous connecter.

  • Saisissez votre adresse e-mail et votre mot de passe dans les cases. Cliquez sur INSCRIPTION.
