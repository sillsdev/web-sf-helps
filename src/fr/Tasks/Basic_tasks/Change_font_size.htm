Modifier la taille de la police

  1. Choisissez un projet.

  2. Dans le volet Navigation, cliquez sur Vérification par la communauté.

  3. Cliquez sur le livre ( <name>).

  4. Sur la barre d'outils, cliquez sur (taille de la police).
