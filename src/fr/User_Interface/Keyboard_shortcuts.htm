Conseils

  • Scripture Forge fonctionne dans un navigateur Internet. Les navigateurs vous permettent d'ouvrir un hyperlien dans un nouvel onglet.

Dans Scripture Forge, pour ouvrir une zone dans un autre onglet, vous pouvez appuyer sur la touche Ctrl puis cliquer sur une zone dans le volet de navigation.

Vous pouvez également cliquer sur une zone dans le volet de navigation avec le bouton de souris du milieu (qui est souvent la roulette de défilement).
