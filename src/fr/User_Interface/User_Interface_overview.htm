Vue d'ensemble de l'interface utilisateur

L'interface utilisateur (IU) est ce que vous voyez quand Scripture Forge est ouvert dans un navigateur Internet.

Voici quelques éléments que vous voyez et utilisez dans Scripture Forge, suivant votre rôle :

  • Barre d'application supérieure

  • Volet de navigation

  • Barre d'outils

  • Boîtes de dialogue

  • Raccourcis clavier

  • Choisir une langue d'interface utilisateur.

  • Redimensionner les volets dans la fenêtre Scripture Forge :

Par exemple, vous pouvez vouloir le faire lorsque vous regardez les questions et réponses ou les commentaires sous le texte d'un livre.

    1. Déplacez le curseur de votre souris sur les bords entre les volets.

    2. Si votre curseur se change en flèches avec des lignes de séparation (), cliquez et faites glisser le bord d'un volet pour modifier les tailles relatives de ces volets dans la fenêtre.
