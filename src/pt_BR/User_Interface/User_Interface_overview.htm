Visão geral da Interface do Usuário

A interface do usuário (IU) é o que você visualiza quando o Scripture Forge está aberto num navegador da Internet.

Aqui estão alguns itens que você visualiza e usa no Scripture Forge, dependendo de sua função:

  • Barra de Aplicativos Superior

  • Painel de Navegação

  • Barra de ferramentas

  • Caixas de diálogo

  • Atalhos do teclado

  • Selecionar um idioma da interface do usuário.

  • Redimensionar os painéis na janela do Scripture Forge:

Por exemplo, talvez você queira fazer isso enquanto visualiza perguntas e respostas ou comentários abaixo do texto de um livro.

    1. Mova o ponteiro do mouse sobre as bordas entre os painéis.

    2. Se o cursor mudar para setas com linhas divididas (), clique e arraste a borda de um painel para alterar os tamanhos relativos desses painéis na janela.
