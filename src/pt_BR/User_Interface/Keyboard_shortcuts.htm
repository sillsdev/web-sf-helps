Dica

  • O Scripture Forge é executado num navegador de Internet. Os navegadores permitem que você abra um hiperlink numa nova guia.

No Scripture Forge, para abrir uma área em outra guia, você pode pressionar a tecla Ctrl e, depois, clicar numa área do painel de Navegação.

Você também pode clicar numa área do painel de Navegação com o botão do meio do mouse (que geralmente é a roda de rolagem).
