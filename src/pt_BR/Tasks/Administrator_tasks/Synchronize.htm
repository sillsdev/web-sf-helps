Sincronizar

Pessoas com a função de Administrador podem executar estes passos para sincronizar este projeto do Scripture Forge com o repositório do Paratext.

  1. Selecionar um projeto.

  2. No Painel de Navegação , execute estes passos:

    • Clique na seta para baixo () e depois clique no projeto desejado.

    • Clique em Sincronizar.

      • Clique no botão SINCRONIZAR no painel principal.

Aparece um indicador de progresso.

As alterações realizadas no Scripture Forge são enviadas para o repositório. As alterações realizadas no Paratext que foram enviadas para o repositório são recebidas no Scripture Forge.
