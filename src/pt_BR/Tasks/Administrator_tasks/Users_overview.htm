Visão geral de usuários

No Painel de Navegação, Usuários é uma das áreas.

Aqui estão as tarefas que você pode realizar nesta área:

  • Compartilhar com outros para convidar pessoas para ajudar como verificadores da comunidade.

  • Filtrar usuários - Para localizar um usuário na lista, digite o nome dele. Para filtrar uma determinada função, digite essa função.

  • Can add & edit questions - For a Paratext user who is not an Administrator, select () this check box if you need to give that person permission to add, edit, import and archive questions.

  • Remove users - To remove a user from the current project, click (Remove from project).

Nota

  • As funções e permissões dos usuários só podem ser alteradas no Paratext (Permissões de usuário em Configurações do Projeto).

Será necessário executar Enviar/Receber no Paratext e depois Sincronizar o projeto do Scripture Forge.

Em seguida, depois que o usuário fizer o login, a alteraçao da sua função será exibida no Scripture Forge.
