Conectar projeto

O administrador deve conectar o projeto primeiro. Depois disso, pessoas com outras funções podem se conectar a ele.

  1. No painel de Navegação, clique na seta para baixo do projeto () e depois clique em Conectar projeto.

A janela Conectar Projeto do Paratext aparece.

  1. Clique na seta para baixo no controle do Projeto do Paratext e depois clique no projeto.

  2. Se você visualizar Configurações Adicionais, selecione () ou desmarque () as caixas de seleção conforme desejado.

Estes itens são descritos em Configurar ferramentas do projeto. Estas configurações podem ser alteradas depois.

  1. Clique em CONECTAR.

Aparece um indicador de progresso. Quando terminar, cada um pode realizar suas tarefas de acordo com sua função.

Importante

  • Configurações Adicionais aparece se o projeto que você deseja conectar ao Scripture Forge ainda não existe no Scripture Forge.

O administrador que concluir os passos neste tópico criará o projeto no Scripture Forge.

 

Depois disso, os tradutores e revisores não verão Configurações Adicionais quando se conectarem ao projeto. Isso ocorre porque foram configurados pelo Administrador.

  • Project types you can connect to:

    • Standard Translation and Daughter Translation

    • Back Translation, Auxiliary, Transliteration (Manual) and Transliteration (using Encoding Converter)
