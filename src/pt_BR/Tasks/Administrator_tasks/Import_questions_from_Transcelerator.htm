Importar perguntas do Transcelerator

Administrators can import questions that are in the Transcelerator plugin.

Pré-requisitos

  • Download and install Transcelerator, if it is not already installed.

  • On the Transcelerator File menu, click to select () Produce Scripture Forge Files.

Execute estes passos:

  1. In Transcelerator, type the translations of the English questions you want to import in the Translation column.
    Then, select the Confirmed check box for those questions. Close Transcelerator.

  2. No Paratext, execute Enviar/Receber o projeto.

  3. No Scripture Forge, sincronize o projeto.

  4. No painel de Navegação, amplie Verificação da Comunidade e depois clique em Visão geral.

  5. Clique no botão IMPORTAÇÃO EM MASSA. It is near the ADD QUESTION button.
    The Import Questions dialog box appears.

  6. Clique em Importar do Transcelerator.

Dica

  • Se você editou perguntas ou deseja importar mais perguntas, execute estes passos novamente.
