Excluir um projeto

  1. Selecione o projeto que você excluirá.

  2. Clique em Configurações no painel de Navegação.

  3. Leia atentamente as informações abaixo de Excluir este projeto.

  4. Sincronize o projeto com o repositório do Paratext (recomendado).

  5. Se você tiver certeza de que deseja continuar, clique em EXCLUIR ESTE PROJETO.

The Are you sure you want to delete this project dialog box appears.

  1. Digite o nome do projeto para confirmar que você excluirá o projeto correto.

  2. Clique em ENTENDO AS CONSEQUÊNCIAS, EXCLUA ESTE PROJETO.
