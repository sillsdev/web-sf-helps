Configurar ferramentas do projeto

Pessoas com a função de Administrador podem realizar esta tarefa.

  • Na Barra de Aplicativos Superior, clique no seu avatar e, em seguida, clique em Página inicial do projeto. Depois clique em Configurações no painel de Navegação.

Ferramentas

Traduzir

  • Select () Translation Suggestions if you want to enable translation suggestions.

Depois, siga estes passos:

    1. Click Based on to see a list of Projects and Resources.

    2. Click the project or resource you want as the basis of the translation suggestions.

Projects are Paratext projects and Resources are from the Digital Bible Library.

The Based on project or resource is the source of the translation suggestions. The books and their layout in the project you are translating must match the books in the Based on project or resource. To do this, in Paratext when you create a book in your project use the Create based on option. This option is in the Create Book(s) dialog box. You open it from the Project menu > Manage books > Create book(s).

When translating, you can use the button to swap the Based on project or resource and your project from one side of the window to another.

  • Share - If selected (), the share button ( ) will appear in Translate. This enables other users, in addition to Administrators, to use the Share with others dialog box.

Then, select () one of these options:

●  Select Anyone with a link if you want allow anyone who has the project link URL to view the project.

●  Select Specific People if you want to only allow a person to view the project by responding to an emailed invitation.

Verificação da Comunidade

  • Select () Community Checking if you want to enable this tool. Then, optionally, you can select () one or both of these check boxes:

    • See other's answers and comments - If selected, people will be able to see other peoples' questions and comments.

    • Share - If selected (), the share button ( ) will appear in Community Checking. This enables other users, in addition to Administrators, to use the Share with others dialog box.

Then, select () one of these options:

●  Select Anyone with a link if you want allow anyone who has the project link URL to join as a community checker.

●  Select Specific People if you want to only allow community checkers to join the project by responding to an emailed invitation.
