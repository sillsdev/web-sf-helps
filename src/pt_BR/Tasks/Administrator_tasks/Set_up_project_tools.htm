Configurar ferramentas do projeto

Pessoas com a função de Administrador podem realizar esta tarefa.

  • Na Barra de Aplicativos Superior, clique no seu avatar e, em seguida, clique em Página inicial do projeto. Depois clique em Configurações no painel de Navegação.

Ferramentas

Traduzir

  • Selecione () Sugestões de Tradução se desejar ativar sugestões de tradução. Em seguida, clique na seta para baixo Com base em () e clique num projeto do Paratext.

O projeto que você seleciona com o controle Com base em é o projeto de origem. O projeto que você está traduzindo é o projeto de destino. Esses dois projetos podem ser movidos (trocados) de um lado da janela para outro. Os livros do projeto Com base em devem corresponder aos livros do projeto de destino. Caso contrário, quando você clicar num livro, pode não haver um livro correspondente no projeto de origem.

Verificação da Comunidade

  • Selecione () Verificação da Comunidade se desejar ativar esta ferramenta. Então, se desejar, você pode selecionar () uma ou estas duas caixas de seleção:

    • Visualizar as respostas e comentários de outras pessoas- Se selecionadas, as pessoas estarão aptas a visualizar perguntas e comentários de outras pessoas.

    • Compartilhar - Se selecionado, este botão (compartilhar) aparecerá em Verificação da Comunidade.  Assim, as pessoas que não têm a função de Administrador podem clicar e, em seguida,enviar um convite por e-mail para outras pessoas.

Em seguida, selecione () uma destas opções:

●  Selecione Qualquer pessoa que tenha um link se desejar permitir que qualquer pessoa que tenha o URL do link do projeto entre como um verificador da comunidade.

●  Selecione Pessoas Específicas se desejar permitir somente que os verificadores da comunidade participem do projeto respondendo a um convite por e-mail.
