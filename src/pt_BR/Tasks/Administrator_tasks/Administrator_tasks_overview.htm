Visão geral das tarefas do Administrador

Uma pessoa com a função de Administrador pode realizar estas tarefas:

Controle

  • Perguntas, respostas ou comentários:

    • Adicionar uma pergunta

      • Gravar ou carregar uma pergunta

      • Work with answers

    • Arquivar ou republicar uma pergunta

    • Conectar projeto

    • Excluir uma pergunta

    • Editar uma pergunta

    • Importar perguntas de uma planilha

    • Importar perguntas do Transcelerator

  • Projeto:

    • Configurar sugestões de tradução

    • Conectar um projeto do Paratext ao Scripture Forge

    • Excluir um projeto

    • Revisar o progresso da tradução

    • Sincronizar

    • Visão geral de usuários

Visualização

  • Revisar o progresso da Tradução
