Fazer login / Sair

Login

  • Depois de cadastrado , e o projeto ter sido conectado, clique em Fazer Login e execute um destes passos:

    • Clique em FAZER LOGIN O COM PARATEXT, e depois siga as etapas das páginasAutorizar Solicitação.

Sua função no projeto do Paratext é a mesma no Scripture Forge se você utilizou FAZER LOGIN COM O PARATEXT pelo menos uma vez.

Depois disso, sua função no projeto mantida na memória se você efetuar o login usando as outras opções de login abaixo.

  • Clique em FAZER LOGIN COM O GOOGLE, e depois siga as etapas das páginas Entrar com o Google.

  • Clique em FAZER LOGIN COM O FACEBOOK, e depois siga os passos das páginasEntrar no Facebook.

  • Click LOG IN WITH PHONE NUMBER.

  • If the country code you see is correct, type your phone number.

  • Otherwise, click the arrow () and search for your country code.  Then, type your phone number.

A code is sent to your phone.

Type that code to log in. 

  • Digite seu e-mail e sua senha nas caixas, e depois clique em FAZER LOGIN.

Sair

  • Para sair, na Barra de Aplicativos Superior, clique no seu avatar e depois clique em Sair.

Importante

  • Todos os dados do Scripture Forge são removidos do seu computador ou dispositivo toda vez que você sair. So, if you want to save changes you made while you were offline () do not log out until you have an internet connection ().
    Changes you make while offline are stored locally and will be sent to the server the next time you have an internet connection. No entanto, esse armazenamento local é removido sempre que você sair do sistema. O logout não exclui os dados do servidor.
