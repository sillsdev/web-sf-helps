Selecionar um livro

  1. Selecione o projeto que possui o livro com o qual deseja trabalhar.

  2. No painel de Navegação, clique em Traduzir ou Verificação da Comunidade.

  3. Execute um destes passos:

    • Clique no livro desejado ( <name>).

    • Em Visão geral, clique no nome do livro que deseja.

  4. Use a barra de ferramentas para selecionar um capítulo.
