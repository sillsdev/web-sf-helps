Abrir o Scripture Forge com o ícone da área de trabalho

Primeira vez

Depois que instalar o ícone da área de trabalho, execute estes passos:

  1. Clique duas vezes no ícone da área de trabalho .

  2. Clique no botão FAZER LOGIN para fazer o login.

  3. Selecione um projeto.

  4. Para trabalhar offline, desconecte-se da internet.

  5. Execute todas as tarefas que não requerem acesso à Internet.

  6. Quando você quiser parar de trabalhar, clique em X (Fechar) ou pressione Alt+ F4 para fechar o Scripture Forge sem fazer logout.

Importante: Se você Sair enquanto estiver offline (), nenhum trabalho feito offline será salvo.

Uso normal

  1. Clique duas vezes no ícone da área de trabalho .

  2. Execute um destes passos:

    • Se você não tiver acesso à Internet e não fez o logout na última vez que você utilizou, o Scripture Forge abre para que possa usar.

    • Se você não tiver acesso à Internet, mas você fez o logout da última vez utilizou, então você precisa ter acesso à Internet para continuar.

    • Se tiver acesso à Internet e for solicitado a fazer o login, então faça o login.

Dica

Num navegador Chrome ou Edge, aqui está uma outra forma para abrir o Scripture Forge:

  1. Faça login no Scripture Forge.

  2. Perto do fim da barra de endereço do navegador da web (também chamada barra URL), clique em (Para abrir este link, selecione um aplicativo).

  3. Na caixa Abrir com, clique em Abrir.

O Scripture Forge abre em outra janela. A guia do navegador fecha automaticamente.
