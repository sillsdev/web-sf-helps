Selecionar um idioma da Interface do Usuário

  • Na Barra de Aplicativos Superior , clique em (Idioma) e depois clique no idioma desejado.

A Interface do Usuário (UI) muda para esse idioma. Se algumas palavras não forem traduzidas, aparecerão em inglês.

Se você não visualizar o idioma que deseja na lista, busque mais ajuda.
