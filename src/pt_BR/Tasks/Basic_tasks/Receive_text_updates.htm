Receber atualizações de texto

As pessoas com qualquer função devem visualizar o conteúdo de texto mais recente.

Você e sua equipe precisam considerar estes passos, e executar com uma regularidade combinada.

  • Como os livros são adicionados ou atualizados no Paratext, alguém deve executar Enviar/Receber no Paratext para manter o repositório do Paratext atualizado.

  • No Scripture Forge, quando alguém conecta um projeto pela primeira vez, o conteúdo mais recente do repositório é recebido.

Em seguida, à medida em que os livros são atualizados no Scripture Forge, a pessoa com a função de Administrador do projeto deve sincronizar o projeto do Scripture Forge com o repositório do Paratext.
