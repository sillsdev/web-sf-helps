Visão geral das tarefas de tradução

Uma pessoa com a função de Tradutor pode realizar essas tarefas:

  • Adicionar um comentário

  • Configurar sugestões de tradução

  • Treinar novamente o Mecanismo de Sugestão

  • Trocar origem e destino

  • Sincronizar

  • Traduzir

Nota

  • Se a caixa de seleção Edição Ativada estiver desativada () no Paratext, você não poderá editar o texto no Scripture Forge.

  • Note tags that came from Paratext also appear in Scripture Forge when synchronized if your role permits it.
