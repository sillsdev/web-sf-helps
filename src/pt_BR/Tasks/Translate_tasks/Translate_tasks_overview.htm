Visão geral das tarefas de tradução

Uma pessoa com a função de Tradutor pode realizar essas tarefas:

  • Configurar sugestões de tradução

  • Treinar novamente o Mecanismo de Sugestão

  • Trocar origem e destino

  • Sincronizar

  • Traduzir

  • Drag and drop text

Nota

  • If the Editing Enabled check box is cleared () in Paratext, you cannot edit the text in Scripture Forge.
