Treinar novamente o Mecanismo de Sugestão

As sugestões estão disponíveis se seu projeto tiver outro projeto ou recurso selecionado no controle Baseado em.

Siga estes passos para treinar novamente o mecanismo de sugestão:

  1. Selecione o projeto.

  2. No painel de Navegação , clique em Traduzir.

  3. Clique na seta () de um livro para visualizar os capítulos com perguntas.

O painel Visão geral de Tradução aparece e exibe o painel Mecanismo de Sugestão.

Você visualiza a qualidade com base no número de estrelas, no número de segmentos treinados e no botão TREINAR NOVAMENTE.

  1. Clique em TREINAR NOVAMENTE.

O painel Mecanismo de Sugestão exibe TREINAMENTO e uma barra de progresso exibe o seu progresso.

Quando terminar, você verá o número de segmentos treinados.

Be aware that if the source text does not have any books in common with the target project, you will not see any effects from using this feature. A message will inform you that translation suggestions are unavailable.

Importante

  • Você não deve precisar fazer isso com frequência, porque aprende enquanto você traduz e o processo de treinar novamente pode levar muito tempo.

Aqui estão as razões que você deve considerar ao treinar novamente o mecanismo de sugestões:

  • Você alterou significativamente a maneira de traduzir, especialmente se algo estiver em mais de um lugar.

  • Você realizou um número significativo de alterações que resultaram de uma verificação do consultor.
