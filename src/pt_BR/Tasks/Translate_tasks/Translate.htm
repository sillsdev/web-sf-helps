Traduzir

Se a caixa de seleção Edição Ativada estiver desativada () no Paratext, você não poderá editar o texto no Scripture Forge.

Todos os livros que foram adicionados no Paratext e enviados em seguida para o repositório do Paratext estão disponíveis no Scripture Forge depois que o Administrador do projeto sincroniza o projeto.

  1. Na Barra de Aplicativos Superior, clique no seu avatar e, em seguida, clique em Página inicial do projeto.

  2. No painel de Navegação , clique em Traduzir.

  3. Clique no livro ( <name>).

Abaixo da barra de ferramentas, você visualiza o livro de destino e de origem (com base em). A versificação foi configurada no Paratext.

  1. Use o controle de capítulo na barra de ferramentas para selecionar o capítulo que deseja traduzir.

  2. Verifique se o teclado que você precisa está ativado.

  3. Clique na caixa correspondente a um versículo.

  4. Se Sugestões de tradução estiver definido para exibir (), execute um destes passos para inserir palavras na caixa:

    • Use atalhos do teclado como Ctrl+0 para inserir uma frase inteira ou Ctrl <number> para inserir esse número de palavras, começando pela palavra atual.

    • Clique na sugestão ou pressione a tecla Enter para aceitar a sugestão.

  5. If no suggestions are presented or Translation suggestions is set to hide (), type the translation of the source project.
