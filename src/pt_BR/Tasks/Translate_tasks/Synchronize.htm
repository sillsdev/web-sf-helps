Sincronizar

Pessoas com a função de Administrador ou Tradutor podem executar estas etapas para sincronizar este projeto do Scripture Forge com o repositório do Paratext. Isto inclui as notas mais recentes que aparecem no Paratext.  You (Admin) might want to review Work with answers.

  1. Selecionar um projeto.

  2. No Painel de navegação, clique em Sincronizar.

  3. Clique no botão SINCRONIZAR no painel principal.

Aparece um indicador de progresso.

As alterações realizadas no Scripture Forge são enviadas para o repositório. As alterações realizadas no Paratext que foram enviadas para o repositório são recebidas no Scripture Forge.

Nota

  • Se você sincronizar um projeto que é baseado em outro projeto, o projeto base também será sincronizado. Assim, se esse projeto base foi atualizado, você receberá as alterações mais recentes.

  • O indicador de progresso tem um botão CANCELAR que você pode pressionar para cancelar o processo.

  • When reviewer leaves a note in the text in Scripture Forge, after a sync, the note in Paratext will be attributed to the Paratext user who did the sync, either an Admin or Translator. There will be not record of the Scripture Forge user who does the sync.
