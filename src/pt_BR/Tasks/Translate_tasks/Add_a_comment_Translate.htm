Add a comment to a verse

This topic is done in the Translate area. People with the role of Admin, Translator or Commenter can add these comments.

If you are a Community Checker, see: Add a comment to add a comment to an answer.

  1. Selecionar um projeto.

  2. No painel de Navegação , clique em Traduzir.

  3. Clique no livro ( <name>).

  4. Click the verse you want to comment on.

  5. Click .

A dialog box appears, with the book, chapter and verse shown in the title bar.

  • Verify they are correct.

  1. Type your comment in the dialog box and then click Save.

That verse now shows a comment icon ().

  • Hold your mouse pointer over that icon to read the comment.

If you are using a phone or smaller device, tap the icon to open the dialog box so you can read it.

Edit a comment

  1. Click or tap the icon () to open the dialog box again.

  2. Click or tap the edit pencil ( ) to put it in the edit mode.

  3. Make the desired edits.

  4. Click Save. 

Nota

  • On a mobile device, a bottom sheet appears that you can use to enter and save a comment.

  • When comments are exported to PT as a note, the name of the commenter is shown as a label on the note.
